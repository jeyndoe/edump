Есть такая задача: создать интернет сеть для передачи конфиденциальных данных для ограниченного контингента пользователей. Как я это себе представляю:
1. Есть один или более сайтов (серверов на чужом хостинге), где хранится таблица пользователей и их IP адресов. Это необходимо для обновления тех IP-адресов которые пользователям выделяются динамически (статические редко когда, но всё же меняются тоже). Передача идёт через шифрование данных, ключи время от времени меняются. Назначение сервера - получение IP-адреса нужного пользователя. Дальнейший обмен идёт P-to-P, то есть - без участия сервера, по крайней мере до потери связи.
2. У каждого пользователя имеется софтина, посредством которой происходит получение IP-адреса нужного пользователя(ей), общение, передача файлов, чат, е-мэйл, вплоть до голосовой и видео связи. Софтина выдается при очной встрече, также ключи и пароли.

Это то что первым пришло в голову. Для реализации желательно начать со свободных исходников примерно похожей системы. Я предполагаю воспользоваться (для начала) таковыми от Миранды (они на С++ что для меня очень желательно). Пока не нашёл скриптов для сервера.

У кого есть какие мнения и предложения?
