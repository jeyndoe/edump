The Silicon Realms Toolworks is happy to announce that SoftwarePassport/Armadillo v6.00 "Beta" test version is now available! The direct download link is www.siliconrealms.com/armadillo_engine.shtml.

New features for v6.00 include:

* Additional protections
* The ability to set up keys to expire a certain number of days after they are created
* A subscription licensing option that allows customers to use a program for a set number of minutes or uses each month until the subscription expires
* Expanded support for USB hardware locking
* Network licensing options in the SoftwarePassport interface
* An option that allows no more than one running copy of a program to display a "key expired" message at a time
* An option for preventing problems related to Address Space Layout Randomization (ASLR) on Windows Vista

For more information on the enhancements and bug fixes in v6.00, see the revision histories in the help files ("SoftwarePassport User Guide" and "Armadillo Help"). In addition, we have created an FAQ page to cover some of the questions you may have about this release, including pricing and grace period discounts. That page is available at:

faq.siliconrealms.com/drm/faq.asp?Action=Q&ID=103

As with all beta versions, we will not make custom builds available until we post the final release version. In the meantime, a test build will be available for downloading via the custom build download page. This test build provides the advanced features of a custom build.

We do not recommend releasing software with beta versions of SoftwarePassport/Armadillo, but if you must do so, please test the protections thoroughly to ensure that we haven't missed a bug.

If there are no major problems, the final release will be on July 8.

Please let us know if you have any questions, or run into any problems with this beta version by contacting our support staff at support@siliconrealms.com

Regards,
The Silicon Realms Toolworks Team
