Бегло ознакомился с архитектурой проца. Очень даже прикольная штука.

У проца есть несколько вариантов бута, в том числе USB Boot. Откуда грузиться процу указывает состояние пинов BOOT_SEL1, PE25 и ADIN1. Все описано в --> PDF <-- на сайте производителя.

В данном девайсе, в нормально состоянии проц судя по всему грузится с NAND, чтобы переключить его в режим USB Boot возможно придется проводить манипуляции с лапками. Хотя вполне возможно, что производителем девайса предусмотрен режим аварийного переключения на USB Boot (например, зажать какую-то кнопку при включении девайса).

На сайте производителя проца также есть тулза --> USB Boot Tool <-- с сыкрами и мануалом, которая позволяет читать NAND в режиме USB Boot.


З.Ы.
--> тут ещё почитай <--
>> Переход в USB boot mode осуществлять при зажатой кнопке «В» + «reset» с последующим подключением к usb-кабелю.

Если выйдет таким методом перевести девайс в USB boot mode и поставить дрова (которые в комплекте с USB Boot Tool), то скорее всего тулза USB Boot Tool (с сайта производителя проца) позволит вычитать NAND.

Тока все делать не спеша и внимательно читать мануалы, неправильными движениями можно повредить прошивку.
