mov bx, [ax]
после переработки получилась такая вот рабочая программка:
CSEG segment
org 100h
Begin:
mov dx, offset data
mov bx,dx
add bx,2
mov ax,word ptr [bx]
int 20h
data db 'ABCD$'
CSEG ends
end Begin
Вообще то она просто пишет в ax "CD", благодаря вам узнал что ax и bx не равнозначны по способам применения. Еще раз большое спасибо!
