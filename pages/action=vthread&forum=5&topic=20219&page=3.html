schokk_m4ks1k
Если ставишь бряк на api (например bp MessageBoxA) - то VMProtect тебя спалит, поэтому снимай бряк с начала функции
Code:
  1. MOV EDI,EDI
  2. PUSH EBP
  3. MOV EBP,ESP
  4. ...


и ставь на ее конец (ret)
Code:
  1. ...
  2. POP EBP
  3. RETN 10
