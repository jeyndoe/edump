WSUnpacker v0.19
Code:
  1. WSUnpacker is a "generic" shelling machine, which functions as follows:
  2. 1, a key automatic shelling. Off the shell to see the current list below
  3. 2, a generic unpacking engine. The shell of the unknown can try using a common engine to shelling shelling
  4. 3, shell type of identification. Able to identify the common variety of shells, including the latest VMProtect, SafeEngine, VProtect, ZProtect etc.
  5. 4, the compiler of the identification. Able to identify the common variety of compilers, including Delphi, BCB, Visual C + +, Masm, MingW32, etc.
  6. 5, file type identification. Able to identify a variety of common file types, not through an extension that includes a signature, signature recognition by a variety of file types. Such as pdf, rar, etc.
