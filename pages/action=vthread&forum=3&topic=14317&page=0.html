  14.       "Remove redundant folders from extraction path" option.
  15.       This option will eliminate redundant archive name based folders
  16.       from extraction path if you unpack an archive with
  17.       "Extract to DestName" context menu command and if archive

  45.       by their numerical value. So files will be sorted as

  56.       dirname folder and ignore its contents. By default, if dirname

  74.       tray icon includes the archive name now. Previously only
