Описание формата контейнера FLV открыто, но кодеки остаются в привате.
Нет какого-либо офиц. компонента или API для работы с ними.

Вот что может вам помочь:
fFMPEG - Video transcode engine which also supports FLV
vixy.net flv2mpeg4 - FLV to AVI/MP4/MOV command line transcoder
flvlib - Python library for dealing with FLV files

Вот есть готовый "телевизор" в GPL исходниках:
www.mythtv.org/detail/mythtv

Конечно, проще всего на лету конвертировать flv в avi и работать с ним, но, к сожалению, появляются некоторые артефакты изображения.
