Вариант "для полного чайника" - это всякие книги от "Тейлор, Мишель, Пенман" - типа дельфи в популярном изложении. Доступно и местами весьма забавно.
Для 5 (или для 4?) - Тейксейра, Пачеко(? второго не помню точно) - двухтомник оригинальной доки, афаик. Очень глубоко даже по сейчашним временам. Привить стиль написания прог, дабы потом стыдно не было.
Для начала "современных" технологий - Бобровский, причем вариант для дельфи пораньше, например для 5; затем для последних (имхо у него для каждой дельфы выходило). Это "на пальцах".
Далее - обязательно(!) - Кэнту, 7 для профессионалов и, можно, 2005 - для постепенного въезжания в .нет на дельфи. Хотя последнее - изврат, лучше на C# переезжать, дельфи умирает. Плюсом Архангельский, но у него скорее справочники, а не учебники, без достаточного понимания будет сложно. Хотя к Кэнту это тоже относится.
