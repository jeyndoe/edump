Корпорация Microsoft планирует на этой неделе обновить свой фирменный механизм проверки подлинности программного обеспечения Windows Genuine Advantage для операционной системы Windows XP Professional, так как она считает, что именно эту версию Windows наиболее часто устанавливают, не имея на это лицензии. В своем блоге во вторник, Алекс Кочис (Alex Kochis) - старший менеджер Windows Genuine Advantage group, заявил, что после обновления, уведомления Windows Genuine Advantage станут такими же, как сейчас в ОС Windows Vista SP1.

А это значит, что пользователи, чьи копии Windows XP не пройдут проверку на подлинность, после перезагрузки ПК увидят пустой черный экран. И хотя пользователи будут иметь возможность переключаться, черный экран будет возвращаться каждые 60 минут до тех пор, пока не будет пройдена проверка на подлинность ОС. Также пользователям будет напоминать о не пройденной валидации фоновая надпись с текстом "Ask For Genuine Microsoft Software".
