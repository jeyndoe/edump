Проблема решена
Написал письмо в Compuware.
Вот что ответили:


Problems with hyperthreading should be fixed if you install 3.2.1 of Driver
Studio. Do both the host and target installs for a single processor setup
and remember to reboot after.

This patch provides several fixes for DriverStudio 3.2 which include item
for Softice and Visual Softice. Below is link to the zip file containing
the patch. Download this file and unzip it to a temp directory, then follow
the instruction below to install.

ftp://ftp.compuware.com/pub/driverstudio/outgoing/patch/DS3.2.1.zip


There are two sections to the install. One for the target machine and one
for the host machine. NOTE: A single machine requires both the host and
target files.

1 Target Machine installation

· Unzip the files to a temporary directory.

· Copy the .sys and .dat files from the <temp
directories>\target directory to the machines system32\drivers directory.

· Copy the appropriate siksym.sys file from the
<temp directories>\target\<machine type> directory to the machines
system32\drivers directory.

2 Host Machine installation

· Unzip the files to a temporary directory.

· Copy the .dat files from the <temp directories>\Host
directory to the machines system32\drivers directory.

· Copy the .dat files from the <temp directories>\Host
directory to the machines Compuware\DriverStudio\VisualSoftICE directory.

· Copy DS.exe from the <temp directories>\Host directory
to the machines Compuware\DriverStudio directory.


3. If your target machine will be using Standard Softice as a stand alone
debugger at some point then you will also need to copy to the
system32\Drivers directory the following two files that are located in the
target subdirectory of this patch.

NTICE.SYS
CPTHOOK.SYS

Customer Technical Support Analyst
