Ладно. Учитывая молчанку.
Для регистрации стандартный бряк на считывание текста. Дальше исправление xor eax,eax на mov ah,1.
Уничтожение ограничения символов: bpx GetDlgitemInt - ведь должна же прога считать длину символов для пароля. Видим куда заносится результат. bpm аддресс. Видим фокус с div ecx. Понимаем вот почему есть ограничение - метод остатка. Ищем что за переменная, бряк на запись, исправляем mov аддресс,6 на
любое число (сколько есть совести ;) ). Всё.
Вопрос: неужели нужно вичислять серийник ? По большому счёту можно два биты исправить.
Сама прога: www.elcomsoft.com/archpr.html
