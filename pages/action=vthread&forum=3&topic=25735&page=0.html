CyberChef - это простое, интуитивно понятное веб-приложение для выполнения всевозможных «кибер» операций в веб-браузере. Эти операции включают в себя простое кодирование, такое как XOR или Base64, более сложное шифрование, такое как AES, DES и Blowfish, создание двоичных и шестнадцатеричных кодов, сжатие и распаковку данных, вычисление хэшей и контрольных сумм, разбор IPv6 и X.509, изменение кодировок символов и многое другое.
Этот инструмент предназначен для того, чтобы технические и нетехнические аналитики могли манипулировать данными сложными способами, не сталкиваясь со сложными инструментами или алгоритмами. Он был задуман, спроектирован, построен и постепенно улучшен аналитиком за 10% инноваций за несколько лет.

ps: CyberChef создан сотрудниками GCHQ — спецслужба Великобритании, ответственная за ведение радиоэлектронной разведки и обеспечение защиты информации органов правительства и армии.

9.7.0 - 2019-09-13
- 'Optical Character Recognition' operation added
- 'Bacon Cipher Encode' and 'Bacon Cipher Decode' operations added
- Various Steganography operations added: 'Extract LSB', 'Extract RGBA', 'Randomize Colour Palette', and 'View Bit Plane'
- 'Render Markdown' operation added
- 'Show on map' operation added
- 'Parse UDP' operation added
- 'Parse SSH Host Key' operation added
- 'Defang IP Addresses' operation added
- Multiple inputs are now supported in the main web UI, allowing you to upload and process multiple files at once
- A Node.js API has been implemented, meaning that CyberChef can now be used as a library, either to provide specific operations, or an entire baking environment
- A read-eval-print loop (REPL) is also included to enable prototyping and experimentation with the API
- Light and dark Solarized themes added
