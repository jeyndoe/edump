может посоветуете нужные туторы?
HEX-редактором нужно пользоваться в следующих случаях :
1) Ты хорошо представляешь внутреннюю структуру файла и хочешь, либо уточнить какие-то моменты, либо слегка подредактировать.
2) У тебя есть четкая инструкция, как сделать патч.
HEX-редакторы вещь историческая : сделать его просто и относительно несложно пользоваться. Был полезен во времена, когда отладчиков с возможностью патча не было. Сейчас есть олли и есть несколько "автоматизаторов" патча. Увы автоматизаторы предназначены для сложных вещей и сделать патч одного байта для новичка непросто. Патч в олли предназначен для "распакованных" файлов и тоже вводит новичков в "непонятки". Есть например в олли плаг для Dup2, но мне проще делать ручками.
Поэтому и для новичков и для профи HEX-редактор вещь по прежнему незаменимая
