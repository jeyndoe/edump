Согласно сообщению, опубликованному в блоге VoIPWiki, китайской фирме удалось провести обратный инжиниринг протокола, используемого сервисом Skype. Это позволит ей создать собственное клиентское ПО для осуществления телефонных звонков через VoIP-инфраструктуру Skype.

В настоящее время каждый ПК с инсталлированным ПО Skype может быть использован в качестве ретранслятора данных, передающихся между двумя другими компьютерами. Альтернативный клиент не поддерживает режим Supernode, что, если он распространится достаточно широко, может оказать негативное воздействие на качество сервиса Skype.

Пока китайская версия Skype обеспечивает лишь одноранговую телефонную связь между ПК, в дальнейшем компания рассчитывает расширить его функциональность добавив обмен мгновенными сообщениями и информацией о состоянии абонентов.
