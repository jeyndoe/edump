Valemox пишет:
Чота ничо подобного там не нашел.
Шапка по X4:
1. Delete PSIClient.dll in the Corel PhotoShop Program, or rename PSIClient.dll to a different suffix, such as PSIClient.old.
In the below, win+R refers to the keyboard "Windows logo" key + the R key, which quickly brings up the Run command:

2. win+R -> services.msc
Stop/Disable ProtexisLicensing
In step 2, the name of the service to stop is now Protexis Licensing V2

3. win+R --> sc delete ProtexisLicensing
In step 3, the name of the service to pass to sc for deletion is now PSI_SVC_2

4. In the /SYSTEM32 folder:
-delete PSIService.exe
-copy PSIKey.dll
-paste in Program Dir
-rename to PSIClient.dll

0. Полностью закрыть программу

1. Удаляем или переименовываем файл PsiClient.dll, например в PSIClient.old. Файл лежит здесь:
c:\Program Files\Corel\CorelDRAW Graphics Suite X4\Programs\PsiClient.dll
2. Останавливаем службу Protexis Licensing V2
3. Удаляем файл PSIService.exe из ProgramFiles\Common Files\Protexis\License Service
4. Перемещаем файл PSIKey.dll в c:\Program Files\Corel\CorelDRAW Graphics Suite X4\Programs\ и переименовываем его в PSIClient.dll.
5. Удаляем папку Protexis из ProgramFiles\Common Files
