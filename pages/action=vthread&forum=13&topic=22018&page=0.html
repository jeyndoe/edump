Программа для извлечения файлов из файлов-контейнеров созданных при помощи Enigma Virtual Box.

Возможности:
- Показывает информацию о том с какими опциями был создан контейнер и размер данных которые там находятся.
- Извлекает файлы, сжатые распаковывает.

Планы на будущее:
- Извлечение реестра
- Извлечение с внешних пакетов

Написано на питоне, опенсорс.

Заметка в блоге: http://reu.org.ua/ru/tools/my-tools/enigma-virtual-box-extractor.html
Проект на github'e: https://github.com/reu-res/evb-extractor
Для получения последних обновлений чекайте ветку dev.
