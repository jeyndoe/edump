_ftp://ftp.compuware.com/pub/driverstudio/outgoing/patch/DS3.2.1.zip

There are two sections to the install. One for the target machine and one
for the host machine. NOTE: A single machine requires both the host and
target files.

1 Target Machine installation

· Unzip the files to a temporary directory.

· Copy the .sys and .dat files from the <temp
directories> arget directory to the machines system32drivers directory.

· Copy the appropriate siksym.sys file from the
<temp directories> arget<machine type> directory to the machines
system32drivers directory.

2 Host Machine installation

· Unzip the files to a temporary directory.

· Copy the .dat files from the <temp directories>Host
directory to the machines system32drivers directory.

· Copy the .dat files from the <temp directories>Host
directory to the machines CompuwareDriverStudioVisualSoftICE directory.

· Copy DS.exe from the <temp directories>Host directory
to the machines CompuwareDriverStudio directory.


3. If your target machine will be using Standard Softice as a stand alone
debugger at some point then you will also need to copy to the
system32Drivers directory the following two files that are located in the
target subdirectory of this patch.

NTICE.SYS
CPTHOOK.SYS

Customer Technical Support Analyst
