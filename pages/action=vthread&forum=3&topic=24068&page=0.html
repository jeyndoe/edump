BinDiff используется для анализа и сравнения бинарников. Данная тулза пользуется совершенно заслуженной популярностью у экспертов по информационной безопасности и применяется для поиска малвари, анализа патчей и других похожих задач. BinDiff поддерживает архитектуры x86, MIPS, ARM/AArch64, PowerPC и так далее.

Для использования BinDiff понадобится IDA Pro не ниже версии 6.8.
