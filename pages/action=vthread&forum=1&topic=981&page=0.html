есть распаковщик картинок для этих игрушек вплоть до версии 4.3. IMO больше ничего не нужно. А регистрация там простейшая - есть фиксированный набор ключей, в программе прошит массив хешей от ключей. После ввода ключа он хешируется и значение сравнивается со списком. Если есть - код правильный. А далее самое главное - из регкода достается ключ расшифровки картинок, требующих регистрации. Этот ключ одинаков для всех возможных регкодов. Прошлую версию я закейгенил благодаря наличию в проге блеклиста, из которого я достал ключ расшифровки, а для этой ключ вытащили из легального регкода. В принципе можно и все возможные ключи сформировать но лениво, ибо картинки итак можно посмотреть, а игры сами по себе там никакие.
