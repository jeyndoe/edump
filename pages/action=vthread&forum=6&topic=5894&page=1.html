2B XOR 23 == 2B - 23 => 08
2B XOR 2A == 2B - 2A => 01
B4 XOR 34 == B4 - 34 => 80

2B XOR 2A == 2B / 2A => 01
37 XOR 36 == 37 / 36 => 01
B5 XOR B4 == B5 / B4 => 01
B7 XOR B6 == B7 / B6 => 01
D9 XOR D8 == D9 / D8 => 01

2B XOR 40 == 2B + 40 => 6B
33 XOR 48 == 33 + 48 => 7B
B4 XOR 40 == B4 + 40 => F4

36 XOR 6A == 36 * 6A => 5C
B4 XOR E4 == B4 * E4 => 50
DE XOR A2 == DE * A2 => 7C
E0 XOR E0 == E0 * E0 => 00
E2 XOR 9E == E2 * 9E => 7C
