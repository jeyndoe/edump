Это не крякерский метод ;)

В каталоге с 98 есть файл: MSBATCH.INF
В данный раздел нужно добавить строку:

[Setup]
Express=1
InstallDir="d:\windows"
CleanBoot=0 ; <<<< ВОТ ЭТУ

тогда 98 не трет загрузочный сектор, а в boot.ini добавляй:
[operating systems]
C:\="Microsoft Windows 98"
