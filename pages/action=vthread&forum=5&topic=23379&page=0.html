romfiles - это и есть бд TransBase, насколько я знаю.

Попробуйте выполнить
Code:
  1. select t.tname
  2. from systable t
  3. join syscolumn c on t.segno = c.tsegno
  4. where c.ctype = 'blob'


Получите список таблиц, у которых есть столбец с типом данных BLOB. В этих таблицах возможно и будут нужные файлы
