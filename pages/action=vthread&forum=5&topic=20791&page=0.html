причём какая функция при этом вызывается в программе я незнаю.
Ну, в вот в этом-то и вся проблема!
При нажатии кнопки происходит Event и посылается Message into system message Queue.
Ваша программа эти messages из system queque извлекает и пропускает через message pump. Из него они идут в WndProc.
(Message Pump: GetMessage, TranslateMessage, DispatchMessage) to --> WndProc().
WndProc() обрабатывает messages как вы пожелаете.
Там его (этот Message ) и надо ловить (in Message Pump).
Каждый message имеет специфический идентификатор, по которому видно откуда он идет и в чем его суть. Все это подробно описано у Чарльза Петзольда, да и вообще полно туториалов по интернету.
