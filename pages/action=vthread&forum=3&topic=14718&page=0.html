Отладчик для 32-х и 64-х битных приложений.



* Написан на C.
* Отладчик пользовательского режима (не режима ядра!)
* Интерфейс SoftICE.
* Отрисовка интерфейса при помощи www.libsdl.org/ (Simple DirectMedia Layer)
* Поддержка pdb при помощи DIA (Debug Interface Access)
* Дизассм при помощи ragestorm.net/distorm/

ugdbg.sourceforge.net/
svn: https://ugdbg.svn.sourceforge.net/svnroot/ugdbg
