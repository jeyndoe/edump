Здравствуйте!

Могли бы подсказать как решить следующую задачу.
Мне необходимо создать инсталяционный пакет для Windows-приложения, который будет находиться на компакт диске. Данный пакет будет содержать программу для выбора и отображения обучающих курсов. Каждый курс состоит из нескольких файлов. На компьютер программа должна будет устанавливаться только один раз. На завершающем этапе мастера необходимо будет провести активацию для установленной программы. Это будет LiveID и соответственно его пароль. Далее пользователю необходимо будет отправить данные на сервер и при соответствии введенных данных сервер должен будет вернуть ключ, который необходимо будет поместить на локально созданный файл/файлы располагающийся в какой-либо системной папке. Также стоит вопрос относительно защиты файлов, возможно ли защитить установленные на компьютер файлы. Программа должна будет устанавливаться на 1 год с момента активации. После чего для запуска программы необходимо снова пройти активацию. Предполагаю реализовывать данную задачу в Visual Studio 2010.
