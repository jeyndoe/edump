Есть блокированый Sony Ericsson (DB2020, CID52).
Понятное дело нужно разблокировать
Нашёл один онлайн сервис, называется Davinci, механизм такой:

1. они дают тебе прогу
2. прога из мобилы берет информацию нужную
3. отправляешь это им
4. они на кластере генерируют какой-то MAP файл за 2-4 дня
5. шлют тебе, кормишь этот файл проге она разблокирует мобилу

Это называется полный анлок. При последующей перепрошивке телефона он (телефон) всё равно остаётся разблокированной.
Всё просто отлично но за одну мобилу они хотят 50 евро, не мало.
www.davinciteam.com/davincinew/faqinfo?faq=22

Есть второй способ (но такой способ почему-то для моей трубки никто _пока_ не предлагает), просто пропатчить прошивку и прошить ее. Один парень сказал что такой дешевый способ пока не предлагают чтобы не обрушать рынок Им конечно по 50 евро выгоднее
