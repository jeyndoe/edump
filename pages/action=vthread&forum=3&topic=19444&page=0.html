Imports Fixer v1.6
Code:
  1. - Hex Editor loading data speed improved (whole hex editor recoded).
  2. - Bug fixed while attempting to change a byte in the hex editor in a non writable memory page.
  3. - Bug fixed while redirecting APIs to wsock32.dll.
  4. - RtlSetLastWin32Error is now correctly forwarded.
  5. - 4 ntdll APIs forwarded to user32.dll for VISTA & Windows 7.
  6. - Bug fixed when cancel is chosen in the savedialog of update form. IF just used to ignore the user decision.
  7. - Fixed small bug when IF updated version name is displayed in the savedialog form.
  8. - Weird processes appearing as ??????? in the processes list or those having no path are removed from the list.
  9. - A reminder messagebox pops up if the user tries to select an already terminated process (I noticed it was removed).
  10. - Loading dlls is now done using mapping apis instead of LoadLibraryEx.
  11. - Critical bug fixed while opening processes on x64 machines.
  12. - Bug fixed when trying to set BaseOfCode of WinUpack packed targets.
  13. - LoadConfig and COM are now fixed in the PE header.
  14. - Dumper automatically displays data when pressing the arrows in order to improve speed.
  15. - About section filled.
  16. - IF update is redirected to IF official download site and no longer uses direct downloads.
  17. - IF is no longer packed to avoid false virus alerts.
  18. - IF Help updated and is now in pdf format.
  19. - Using backspace in hex edit boxes now allowed.
  20. - Original byte in Bytes Editor removed.
  21. - IT & IAT Tab is henceforth disabled until a process is selected.
  22. - Added the dates of new released versions in history.
