Со словами "е* Вашу мать,AMD и мать Вашу е*" я, всё таки установил новый амдэшный драйвер на убунте 15.10
Спешу написать гайд, ибо может пригодится!
Оригинал тут: http://www.taringa.net/post/linux/19124439/AMD-Radeon-Crimson-Driver-en-Ubuntu-15-10.html

1. Естественно, перед началом установки должен быть установлен обычный (и кстати, самый лучший, в отличие от пропиетарного г-на) стандартный open-source драйвер-враппер. Если Вас не интересуют навороченные стимовские игры - тогда лучше оставить его!
Code:
  1.     sudo apt-get purge "fglrx.*"
  2.     sudo rm /etc/X11/xorg.conf
  3.     sudo apt-get install --reinstall libgl1-mesa-glx libgl1-mesa-dri libgl1-mesa-glx-lts-willy
  4.     sudo dpkg-reconfigure xserver-xorg
  5.     apt-get install --reinstall xserver-xorg-core
  6.     sudo reboot //все таки желательно


2. Качаем отсюда wget http://www2.ati.com/drivers/linux/radeon-crimson-15.11-15.30.1025.zip или заходим --> на эту страничку <--

3. Для драйвера нужны linux-хеадеры и ещё куча новых пакетов. апт-гет инсталлим
Code:
  1. sudo apt-get install cdbs dh-make dkms execstack dh-modaliases linux-headers-generic libqtgui4 xserver-xorg-dev
  2. sudo apt-get install lib32gcc1 mesa-common-dev

3.1 Создаем символьную ссылку на version.h
Code:
  1. sudo ln -s /usr/src/linux-headers-4.2.0-18-generic/include/generated/uapi/linux/version.h /lib/modules/4.2.0-18-generic/build/include/linux/version.h


4. Если тупо запустить установщик, то он не захочет ничего устанавливать! В файле /usr/share/ati/fglrx-install.log будет много разного мусора, поэтому три раза плюем на него и набираем в консоли для генерации пакетов:
Code:
  1. sudo ./amd-driver-installer-15.30.1025-x86.x86_64.run --buildpkg Ubuntu/wily

Ждем пару минут и в консоли опять высвечивается куча ошибок, но нам плевать - ведь пакеты сгенерированы успешно!

5. Переходим в tmp каталог:
Code:
  1. cd /tmp/
  2. ls *.deb


что там лежат deb файлы:
fglrx_15.300-0ubuntu1_amd64.deb
fglrx-amdcccle_15.300-0ubuntu1_amd64.deb
fglrx-core_15.300-0ubuntu1_amd64.deb
fglrx-dev_15.300-0ubuntu1_amd64.deb

6. Устанавливаем пакеты!
Code:
  1. sudo dpkg -i *.deb


Отступление: На данном этапе у меня возникли проблемы из-за коряво удаленного пропиетарного драйвера, который был установлен через "программы и обновления":
/tmp$ sudo dpkg -i *.deb
(Чтение базы данных … на данный момент установлено 252890 файлов и каталогов.)

Как вариант, я воспользовался synaptic (раздел состояние) и снес пакеты fglrx со всеми зависимотями.

7. Не забываем!
Code:
  1. sudo amdconfig --initial -f


8.
Code:
  1. sudo reboot


Ради тестов я скачал доту2 реборн: не знаю кто-там говорил на 111% увеличенную производительность, помоему это дешевый пиар! Всё равно очень слабо - картинка временами идёт БОЛЬШИМИ РЫВКАМИ или немного дергается, на воде появляются артефакты. Если не елозить мышкой туда сюда - то вроде всё нормально, fps чистый ~60.
