Вижу 3 возможных глюка :
1. Не тот OllyDump (нужен 2.21.108) -- см. About совего плагина.
2. После запуска OllyDump не убрана галка Rebuild Inport -- LordPE повторяет таблицу импорта
3. LordPE неправильно настроен: должны стоять галки (и только эти) FullDump: Pasete header from disk и ullDump: FixHeader.

О- - и еще -- в lordPE тоже есть галка Rebuild Import Table -- ее тоже снять %)
