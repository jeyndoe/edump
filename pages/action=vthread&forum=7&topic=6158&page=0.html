bitmaster
если в исходниках, то распаковываешь архив:
tar xvf any.tar если tar
gunzip any.gz если gz
bunzip2 any.bz2 если bz2
Это чаще всего встречающиеся архивы.

Потом читаешь readme и если там нет ничего интересного по компиляции и установке, то:
.\configure
make
make install

Если rpm, то rpm -ihv any.rpm. Также встречаются deb архивы, как их устанавливать не в Debian Linux я так и не понял.
