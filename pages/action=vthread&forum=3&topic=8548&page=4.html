SweetScape 010 Editor v10.0 (December 6th, 2019)
Code:
  1. A full debugger is now available for finding and fixing problems with Templates and Scripts.
  2. The debugger can be accessed using the Debug menu and includes stepping, breakpoints, watches and a call stack.
  3. Templates and Scripts are now threaded, meaning other editing operations can be done while a Template or Script is running.
  4. When using the Text Editor, line numbers and ruler labels are now hidden by default (they can be shown with 'View > Addresses > Show Addresses' or 'View > Ruler > Show Labels').
  5. When line numbers or ruler labels are hidden, hover the mouse over the address column or ruler for a second to display a hint popup with the hidden information.
  6. 'View > Tabs/Whitespace > Show Whitespace' now can be used to visualize linefeed types for each line.
