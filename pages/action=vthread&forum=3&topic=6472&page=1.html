ПО MojoPac, которое разработала фирма RingCube Technologies, позволяет переносить персональные настройки настольного компьютера, пользовательские программы и данные на миниатюрные устройства, оборудованные флэш-памятью -- USB-флэш, мобильные телефоны, КПК, аудиоплееры и т.п.

"Вы носите вашу цифровую суть с собой на любом портативном записывающем устройстве", - отмечает Шан Аппаджоду (Shan Appajodu) исполнительный директор и сооснователь RingCube.

При подключении гэджета к ПК под управлением Windows XP, на его экране открывается готовый к работе виртуальный десктоп пользователя. Компания утверждает, что все действия в этой среде не оставляют следов (например, в виде кэшированных файлов) на компьютере-носителе.

Замечания:

- Удобно MojoPac установить на TrueCrypt- диск ( www.truecrypt.org/ )

- Пока работает только на Windows XP (собираются добавить Win2k и 9x)

- Замечено что MojoPac устанавливается только под английской виндой ХР, да и то не под каждой, у него криво реализована проверка версии ОС, поэтому зачастую не признает даже ХР.

- тем не менее установленную под английской ХР удалось запустить и под русской

- Привязка идет к серийному номеру USB-утсройства (именно поэтому несовместим с USB 1.0, а вот TrueCrypt серийник возвращает и на нем пока работает, однако это вызвало почему-то недовольство разработчиков :s6

The update includes the following:

1. Support non-standard Windows install location: users will be able to install and
run MojoPac on a host PC where Microsoft® Windows is installed in a place other than
c:windows. Note: the update from the previous version of MojoPac should be done on a
host where Windows is installed in a standard c:windows directory.
2. Enable/Disable product key: this allows users to disable the MojoPac product key
before enabling it on a new or a reformatted USB device.
3. Support UNC path: to access a network resource using a hostname or IP address
within MojoPac.

driver inside MojoPac. Please see the "How to Enable PunkBuster Support" FAQ on the
MojoPac Forum for details.
5. Fixes for the Recycle Bin issue. Deleted files and folders can now be located in
the Recycle Bin.
6. Fixes for Windows® Media® Player. Media Player 10 can be installed in MojoPac
while a host is installed with Media Player 11.
7. Fixes for the Windows® Internet Explorer® 7 usability problem.
8. Fixes for Domain user related issues including missing entries in "All Programs"
menu and shortcut issues have been resolved.
9. Fixes for Microsoft® Outlook password caching.
