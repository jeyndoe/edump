Привет. Все пишут о том, как прекрасно использовать vpn, proxy, socks, но позабыли о такой старой, проверенной вещи, как ssh туннель.
Вкратце моя безопасность: это случайный из сотни ssh сервер.
В день я делаю несколько подключений к интернету для "ведения дел".
Если вы читаете это - скорее всего, Вы тоже "чем то" занимаетесь в интернете, а значит Вам небезразлична Ваша безопасность.
SSH доступы покупаю здесь: ssh доступы(пакет 50\100шт хватит на несколько лет) , покупаю там потому что саппорт всегда поможет и расскажет, идея возникла совместно.
Плюсы SSH:
Зашифровать можно как 1 порт , так и весь с помощью proxifier.
Я шифрую 1 порт, через него прокидываю icq, jabber и браузер, остальная информация идёт напрямую.
"Рабочий" траффик шифруется и прокидывается через сервер, а игры, например css,wow, bf - идут напрямую, поэтому нет задержек как в случае с vpn, разве это не клёво?
Если интернет обрывается - подключение к ssh тоже обрывается и следовательно траффик не уйдёт напрямую, что является несомненным плюсом.
Плюсы написал
Теперь о настройке:
1) Покупаем доступы: ssh магазин, да, это реклама
2) Скачиваем plink: plink - клиент для подключений ssh.
3) Создаём папку, ложи туда plink и следующие .bat файлы:
Создаём .bat файлы со следующий содержанием:
uote
Содержание:
{SSH_IP} - IP ssh сервера
{SSH_LOGIN} - Логин ssh сервера
{SSH_PASSWORD} - Пароль ssh сервера
{LOCAL_PORT} - Локальный порт через который будет "пускатся" траффик. Указываем нестандартный, любой, например, от 2000 до 65536.

Далее в настройках браузера, icq, jabber ставим в настройках подключения указываем прокси сервер, выбираем следующие данные:
Тип прокси: socks5
ip сервера: localhost
порт сервера: указываем тот что в {LOCAL_PORT}
Логин, пароль - отсутствуют.

Следующим скриптом(За плохой код просьба не ругаться - писался для себя на скорую руку) можно создать .bat файлы:
[PHP]<?php
/*
http://ssh.rentshop.org

Readme:
Кладём в файл $data( ssh.txt ) ip;login;password
Ставим 777 файлу $socks_txt( socks.txt )
Ставим 777 папке $dir_ssh ( ssh )
Заходим на /ssh.php.
Также скрипт можно переделать и использовать в следующем преднзначении:
Берём несколько ssh серверов и вешаем прокси на случайные порты, после у нас есть несколько socks5, сделанных из ssh - можно закинуть в софт для брута, спама, взлома и пр.



*/
//config
$data = file('ssh.txt');
$socks_txt = 'socks.txt';
$dir_ssh='ssh';//Папка куда класть SSH_$port.bat
$port = '44444';//{LOCAL_PORT}
//script
$fps=fopen($socks_txt, 'a+');
if(!file_exists($dir_ssh.'/')) { echo 'Папка <b>'.$dir_ssh.'</b> не создана'; } else{
$x=1;
$count=count($data)-1;

echo'<center>Всё готово, смотрите '$dir_ssh'<b> '.$dir_ssh.'</b></center>';

while($x<$count) {
$x++;
list($ip, $login, $pwd) = explode(";", $data[$x]);
$pwd = str_replace(array("\n\r", '\n\r', "\n", '\n'), "", $pwd);
$random = rand(1,50000);
$fwr = "plink.exe -ssh $ip -C -N -l $login -pw $pwd -D localhost:$port";
$fp=fopen("$dir_ssh\ssh_$random".'.bat', 'a+');
fwrite($fp, $fwr);
fclose($fp);

echo "localhost:$port\r<br>";
fwrite($fps, "localhost:$port\r");
}
echo 'Или можете забрать <a href="'.$socks_txt.'">.txt</a>';
}
fclose($fps);[/PHP]
