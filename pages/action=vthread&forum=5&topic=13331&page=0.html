SVLab пишет:
Ты проверял настройку "Options - Debugging options - Security - Save yser data outside any module to main .udd file" у той Оли, которая не сохраняет бряки в системной dll Да. У экземпляра OllyDbg, сохраняющего информацию (адрес бряка, условие (в моём случае нулевое), информацию, которую необходимо вывести в log data) по условным точкам останова с логированием в файле *.udd, эта опция отключена. Включение этой опции ничего не даёт.
Тот экземпляр OllyDbg, что не сохраняет информацию по брякам тоже проверен со включенной и отлюченной такой опцией. Не сохраняет.
