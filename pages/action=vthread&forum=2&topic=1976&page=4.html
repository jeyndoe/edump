Ну значить есть там такая вещица. File -> Import
Значит импортировать лучше всего уже рабочую модель, например ту что я прикрепил.
Моделька импортируется без проблем.
Теперь делаем такое: File -> Export. Тип файла ставим например GTA San Andreas .DFF (*.dff). Ставим галочку Lock the File. Имя файла не меняем, лучше всего писать в соседнюю папочку
Файл экспортирован.
Создаем новую сцену без сохранения предыдущей.
Выбираем наш экспортнутый файл. Кнопочка Импорт уже неактивна! Вот собственно и все, фвйл залочен.
Залочить можна любую модель для игры, формат которой поддерживает ЗМоделер 2. На игре это не сказывается сообственно никак, но вот модель уже редактировать невозможно.
Я не думаю, что защита (залочивание файлов) там какая-то сильная, но все же не совсем приятно "когда перед носом закрывают двери".
На практике встречался с залочкой DFF для SA и SCS Soft (*.psm) файлов.
