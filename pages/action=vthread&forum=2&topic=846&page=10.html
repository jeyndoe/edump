Здравствуйте, господа (и дамы )

Обращаюсь с просьбой поломать весьма нужные мне прогаммы "basICColor catch 3" и "BasICColor print3c 2.0". У этого поизводителя есть целый цикл программ и если кто-нибудь справиться с этими двумя, то сломать все осталные не составит труда. Особенно нужны две указанные выше.

НАЗНАЧЕНИЕ: "catch 3" - программа получения данных измерений с подключенного к компьютеру прибора (спектрофотометра).
"print3c 2.0" - программа для построения цветовых профилей стандарта ICC на основе измерений полученных в "catch 3"

ССЫЛКИ: "catch"+"print3c" - www.cosys.ru/test/print3c_Installer.zip
файл 14-дневной лицензии - www.cosys.ru/test/basICColor_catch_license.rar
статья с мыслями насчет взлома защиты safecast - www.cosys.ru/test/safecast_info.rar
архив с прогами - 33 метра, остальное - мелочь

ОПИСАНИЕ ОГРАНИЧЕНИЙ: "catch" - 3 типа лицензии (pro - наиболее предпочтительна), отличия в количестве форматов данных на выходе. Лицензия генерится производителем на основании Machine_ID, который можно видеть в меню "о программе". ID не определяется без установленной на компе сетевой карты.

"print3c" - защита (предположительно) Macrovision Safecast. После установки - 5 якобы полнофункциональных запусков. Защита прописывает какие-то данные (demo license code) в unused disk space в начале жесткого диска (?сектор 32), которые, судя по прилагаемой статье не обнуляются даже после полного форматирования. На опыте проверенно, что если восстановить partition с образа Symantec Ghost 8.0, то счетчик обнуляется (!). Система защиты начинает ругаться, что данные нарушены если дефрагментировать раздел (в особенности директории, например, с помощью Executive Software Diskeeper).
Регистрация программы роисходит на основании activation code, который генериться разработчиками на основании machine_ID. При этом ID компа генериться защитой по другому алгоритму, нежели в "catch" - также отсутствую ограничения по наличию сетевой платы.
