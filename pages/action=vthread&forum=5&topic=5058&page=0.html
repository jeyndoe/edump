Сегодня только зарегился на форуме, сегодня же и ставил Soft ICE из Driver Studio 3.2 Что я изменил при настройке(относительно рекомендации о которой ты писал :
1). В Symbol Retriever адрес изменил на :

srv*c:\symserver*http://msdl.microsoft.com/download/symbols

2). Создал директорию snmout в корне C:
3). В symserver распаковал отладочные символы в *.pdb виде, брал их пакетом(195мегов) вот адрес если качать надумаешь:

www.microsoft.com/whdc/DevTools/Debugging/symbolpkg.mspx

(правда там есть для всего, не знаю какие символы тебе нужны, там даже для VISTA BETA есть...)

А вот на пакет ссылка для WinXP_SP2(FULL, ALL Languages) :

msdl.microsoft.com/download/symbols/packages/windowsxp/WindowsXP-KB835935-SP2-slp-Symbols.exe
