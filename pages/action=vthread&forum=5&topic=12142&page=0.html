Потом там ниже есть инструкция по установке другого отладчика. Тоже его скачал по приведённой ссылке, установил (на этот раз файлов никаких не заменяю), жму на файл ntice.bat - опять синее окно с английским текстом. Хоть стой, хоть падай.
__________________________

Наконец, codeview. С него я свои мытарства начал неделю назад и закончил их в сервисном центре.
Этот отладчик упорно не видит файлы, хотя они занесены в его Files Liste. То есть с помощью "Tab" попадаешь в это окошко, выбираешь там файл с расширением .exe потом <Enter>. Всё на этом. Выдаётся акое предупреждение: Сannot find <полный путьк файл>
По включению CodeView этот файл уже занесён в его каталог! И тут же предупреждение, что этого файла нет...

В общем и целом: что же такого в отладчиках, что... никак? FAR, HIEW устанавливаются на "ура" Эти же ни в какую... Буду рад любой помощи. Cпасибо заранее.
