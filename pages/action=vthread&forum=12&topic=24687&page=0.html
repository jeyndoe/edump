  2. #apt-get  install steghide // Установим программу .
  3.  
  4. #mkdir  test  // Cоздаем каталог
  5.  
  6. #cd  test  //  зашли 
  7.  
  8. Создали в test ,photo.jpg  и  source.txt  два файла,

  11. Прячем.
  12.  
  13.  
  14. #  steghide embed  -cf  photo.jpg   -ef  source.txt  // Прячем photo.jpg  <=   source.txt 
  15. Enter passphrase:  пароль для шифрования.
  16. Re-Enter passphrase:
  17. embedding " source.txt "  in   "photo.jpg " ...  done  
  18. #  steghide embed  -sf   photo.jpg  //   вытащим  source.txt
