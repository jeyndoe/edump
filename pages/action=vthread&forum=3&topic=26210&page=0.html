Многофункциональная и настраиваемая в режиме пользователя Win32 и Win64 программа отладки и разбора (дизассемблер). Очень мощный вьювер структуры PE-файлов. Программа действует на уровне команд Intel x86 и впоследствии на самом низком уровне, где ваша программа функционирует. Программа отладки полностью поддерживает выполняемые операции платформы Microsoft .NET и дает возможность равномерно взаимодействовать или функционировать в смешанном режиме отладки. Программа может быть установлена, как программа при запуске, используя ключ параметров Image File Execution Options системного реестра – функция полезна для запуска отладки приложений платформы ASP.NET.
