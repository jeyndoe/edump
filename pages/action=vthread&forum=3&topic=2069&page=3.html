PushTracer will scan the current module locating all PUSH [address] instructions. It will then patch each of these 5-byte instructions with a 5-byte CALL [address]. The call target is a trampoline living within a DLL hosted by the plug-in that will dump the buffer at the PUSH address at run-time, allowing one to monitor the values at all PUSHs within a module.
