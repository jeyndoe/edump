1. Название, производитель, официальный сайт (думаю, без указания версии можно обойтись).
2. Советы и заметки о распаковке

4. Ссылки на туториалы по распаковке и, если есть, автоматические распаковщики.
5. Ссылки на темы по протектору на форуме.
