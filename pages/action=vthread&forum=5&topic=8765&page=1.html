Hi to all
i found in a8b6.dmp this data:
Offset 0 to 3F is dongle cell accesscode data.
offset 41 40 is dongle serior data.
offset 43 42 is dongle ID data.
offset 45 44 is dongle writepassword data.
offset 40 to BF is dongle cell data.
offset c0 to 10bf is first query cell all the data. ( include two cell ,example cell 08 09 )
offset c0 to c3 866FB749 is the response string of query string 00000080, the 866FB749 was written in the file.
offset c4 to c7 297E46D1 is the response string of query string 00000000, the 297E46D1 was written in the file.
offset c8 to cb D49E9834 is the response string of query string 297E46D1,the D49E9834 was written in the file.
offset c8 to cb 20C6C5F9 is the response string of query string D49E9834
next data is same
from c0 to 10bf is 1024 response string datas .
from 10c0 to 20bf is the second query cell data.
All the block size is 0x4000.
1024 * 4 is the one query data size.
