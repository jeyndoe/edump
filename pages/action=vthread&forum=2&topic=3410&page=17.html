Запрос на кряк

[Название программы с указанием версии]
LibMyWitch MySql ActiveX v1.0.13 (component also known as "ActiveMySQL")

[Небольшое описание программы]
Очень крутая ActiveX-компонента для работы с MySQL, в том числе из-за файрволла, т.е. не по стандартным портам, тогда как сам сервер может продолжать работать на стандартном порту, для этого генерится специальный php-скрипт для редиректа данных непосредственно на самом хостинге. Реализован целый ряд других уникальных фичей по работе напрямик с БД MySQL.

[Прямая ссылка на то, откуда можно скачать программу]
www.richtsoft.com/mod.php?theme=richtsoft&mod=download&action=start_download&did=8

[Размер программы]
1,3 Mb

[Пакер\протектор]
Скорее всего незапакован ничем, неопределяется ничего.

[Описание ограничений программы]
Нужен валидный серийный ключ для работы без ограничений. Возможна работа в 2 режимах:
Demo or private key license позволяет свободно пользоваться компонентой около 3 месяцев только на единственной машине локально, для этого режима можно вообще не пользоваться ключом, хотя на сайте и предлагают его бесплатно заказать чисто для учёта спроса. Во время этого периуда появляется nag screen при попытке запуска connect method или иногда вообще просто так (по окончании 3 месяцев). Правильный ключ (2-ой режим) полностью снимает все эти ограничения на локальность использования компоненты (то есть можно слать запросы к MySQL и в инет) и на продолжительность использования, также снимается nag screen.

Серийный ключ следует вводить (присваивать) свойству LicenseKey этого компонента, например:
LibMyWitch.LicenseKey="WAMqKfU842eWZMIoE3dF3Djv5umhQrQHhDV11cyqmk8M8nh x8RvL6ftowfgy2xEdS45DBw72ChqRVxkXDEjnw
YFcOhzbFi4V351"

(этот ключ относится к тестовой категории ключей, см. 1-ую категорию ключей в описании выше).

[Как с вами связаться]
Пишите savchyk(собачка)list.ru
