Grouped Access Tools (GAT) is a very powerful tool that is designed to fight malware, debug applications, software and games, by a means of accessing, editing, analyzing, and manipulating the software. It is able to open processes that are running or start new programs. The user can do very many tasks to the process, such as: suspend, wipe it's memory, terminate, crash, force close, edit it's memory, search for values, and memory dump. Grouped Access Tools has other tools such as running in system mode, running Explorer in system mode, logging processes, and a format converter.

Features
Processes:
- New
- Open
- Find
- Process Scanning (this is able to find hidden processes)
- Find process
- Find DLL/Module
- Multiselection
- Google
- Copy
- Terminate
- Crash
- Force Close
- Suspend/Resume
- Wipe memory
- Dump memory
- Enable All Privileges
- Disable All Privileges
- File Properties
- Go to File's Folder
- Set Normal File Attributes

Memory Editor:
- The addresses are in hexidecimal
- Memory is shown in hexidecimal/decimal and ascii
- Memory is editable
- Find Ascii string
- Find Hexidecimal string
- Write hexidecimal block of memory
- Wipe block of memory
- NOP (No process) block of memory
- Move up and down through memory by threshold bytes
- Go to address
- Go to win32 default header address
- Go to win32 default code address
- Go to End
- Set default threshold (10)
- Set threshold to the number of bytes in view

Value Search:
- search for Integer/Long/Short/Byte values
- search using a range (from address to address)
- Search for increased by values
- Search for decreased by values
- Search for increased values
- Search for decreased values
- Search for changed values
- Search display is updated when values are found
- Search output is in the Address and Value list boxes
- Refresh the output list boxes
- Delete item from output list boxes
- Clear output list boxes
- Add item from output list boxes to definite list boxes
- Add all items from output list boxes to definite list boxes
- New managing list
- Open managing list
- Save managing list
- Delete item from managing list
- Refresh managing list
- Lock managing list item to a value
- Add a custom address to the managing list
- Set description of managing list item
- Set value of managing list item
- Set a range of values in the managing list

Auto File:
- Auto Delete File
- Auto Wipe File
- Delete File on Boot
- Wipe File on Boot
- Move File on Boot
- Take no action on file
- Find File's Accessor
- Save List
- Load List

Auto Process:
- Logs processes that instantiate and decomission
- Logs date, time, process name, and PID
- Auto Suspends process names
- Auto Terminate process names
- Auto Crash process names
- Auto Force Close process names

Process Info:
- Basic Process Information
- Process File Information
- Modules

- Convert from hexidecimal to decimal
