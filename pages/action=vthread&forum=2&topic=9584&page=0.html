Инструкция по эмуляции ключей Hardlock
1. Сдампить ключ программой hl-dump:
_http://www.sporaw.ru/work/files/hl-dump.rar
_http://www.dongla.net/storage/dongles/hl-dump.zip
В архиве находится дампер и полная инструкция по снятию данных с ключа.
2. Скачать солвер для ключей Hardlock:
_http://www.dongla.net/storage/dongles/unidmp2reg.zip
Распаковываем архив с конвертером, подсовываем ему дамп ключа (файл с расширением DAT) и выбираем SafeKey SafeHardlock. Получаем в папке с программой файл *.fs0, который переименовываем в 'mylock.fst'.
3. Скачиваем архив с эмулятором:
_http://rapidshare.com/files/26090550/hl_emul.rar.html
_http://slil.ru/24237472
Пароль на архив: www.dongla.net
Распаковываем, кидаем файл mylock.fst в папку с эмулятором.
4. Обязательно устанавливаем оригинальные драйверы HASP (если снимали дамп ключа не на своей машине):
_http://www.dongla.net/storage/dongles/hdd32.zip
После установки перезагружаем компьютер.
5. Запускаем файл install.bat из папки эмулятора.
После этого программа будет работать без ключа.

Для эмуляции новых ключей Hardlock можно использовать эмулятор от SafeKey. Для эмуляции ключа необходимо проделать следующее:
1. Сдампить ключ программой hl-dump:
_http://www.sporaw.ru/work/files/hl-dump.rar
_http://www.dongla.net/storage/dongles/hl-dump.zip
В архиве находится дампер и полная инструкция по снятию данных с ключа.
2. Скачать солвер (чуток тормозной, зато публичный smile.gif для ключей Hardlock:
_http://rapidshare.com/files/26088885/hl_solver.rar.html
_http://slil.ru/24237428
3. Распаковать архив в корень диска C:\ (обязательно), скопировать дамп Вашего ключа (файл *.DAT) в папку C:\trial_solver_directory_demo_version_only, запустить файл C:\hl_solver.exe, в поле $MODAD указать MODAD Вашего ключа (имя файла с дампом представляет собой нечто такое: 1234.dat, где1234 - MODAD Вашего ключа), и нажать на кнопку Find seeds. Можно попить кофе, посмотреть телевизор, вздремнуть, после чего вернуться к экрану. После успешного завершения работы в правой области программы должна появиться примерно следующая строка:
-> Seed1=0xD5C2, Seed2=0x60DE, Seed3=0xC7A1
Выписавыем её.
4. Скачиваем программу MYLOCK.FST builder:
_http://www.dongla.net/storage/dongles/mylock_builder.zip
Запускаем. В поле Path to Hardlock Dump указываем путь к Вашему дампу. В поля MODAD, Seed1, Seed2 и Seed3 вписываем MODAD Вашего ключа и сиды, полученные солвером. Нажимаем на кнопку Build mylock.fst и получаем необходимый для эмуляции Вашего ключа файл mylock.fst.
5. Скачиваем архив с эмулятором:
_http://rapidshare.com/files/26090550/hl_emul.rar.html
_http://slil.ru/24237472
Пароль на архив: www.dongla.net
Распаковываем, кидаем файл mylock.fst в папку с эмулятором.
6. Обязательно устанавливаем оригинальные драйверы HASP (если снимали дамп ключа не на своей машине):
_http://www.dongla.net/storage/dongles/hdd32.zip
После установки перезагружаем компьютер.
7. Запускаем файл install.bat из папки эмулятора.
После этого программа будет работать без ключа.
