GSS Visual Assembler (последняя на данный момент версия 2.10.5):

--> Офсайт <--

GSS Visual Assembler — среда разработки программного обеспечения для ОС Windows, предназначенная для разработки программ на языке ассемблера.

Удобная визуальная оболочка с легко наcтраеваемым интерфейсом напоминающем интерфейс Borland Delphi или Microsoft Visual Studio. Изначально создавалась под такие ассемблерные пакеты как TASM и MASM32 но есть возможность адаптрировать IDE и под другие пакеты.
Основные характеристики:
- Возможность настроить IDE практически для любого Ассемблерного пакета (TASM, MASM32 и прочих).
- Настройка любой схемы подсветки текста.
- Автоматическое определение процедур, макросов, структур констант и т.д.
- Добавление ресурсов к проекту.
- Компиляция проекта, создание библиотек процедур из DLL (при наличии ассемблерного пакета)
- Максимальная простота управления проектом.
- Автоматичекий поиск декларации переменных.
- Пользовательский интерфейс GSS Visual Assembler максимально копирует привычный
интерфейс Microsoft Visual Studio или Borland Delphi.
- Поддержка всех версий платформы Windows.

GSS Visual Assembler разработан с учотом потребностей, как профессиональных разработчиков, так и начинающих.

Прилагается иллюстрированная справка по использованию:
- Cоздание и управление проектом.
- Настройка компилятора, линкера, отладчика и прочее.
- Вопросы относительно регистрации софта.



Размер: 2.54 MB

--> Полный комплект <--
