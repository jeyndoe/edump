1. Распаковываешь свой файл при помощи Quick Unpack.
2. Обрабатываешь файл программой Resource Binder 2.1 (в настройках убрать последнюю галочку - "Заполнить старую секцию ресурсов нулями").
3. Редактируешь в любом редакторе ресурсов.

А еще лучше для распаковки использовать конкретные распаковщики для конкретных пакеров - они лучше воостанавливают файл, т.к. предназначены именно для распаковки конкретных пакеров.
В твоем случае, надо скачать unFSG 0.9 by smola - unpacking.narod.ru/unpackers/fsg/unFSG_0.9.rar
Распаковать и больше ничего использовать не надо. Полученный файл будет с восстановленными ресурсами и весить на 390 КБ меньше, чем после Quick Unpack'а.
