Code:
  2. char* - C++
  3. PChar - Delphi


сдедующее, твое:
Code:
  2. id:=Ellipsoid.id;
  3. name:=Ellipsoid.ellipsoidName;
  4. radius:=Ellipsoid.EquatorialRadius;
  5. ecc:=Ellipsoid.eccentricitySquared;

а нужно было так:
Code:
  2. Ellipsoid.id:=id;
  3. Ellipsoid.ellipsoidName:=name;
  4. Ellipsoid.EquatorialRadius:=radius;
  5. Ellipsoid.eccentricitySquared:=ecc;
