Нуу... это слишком легко. Например AC_Crackme_01.exe:
Просто по 401A61 адресу пишем 14h и все - нага нет

Идем дальше AC_Crackme_01_A.exe:
Суть одинакова по 401E01 адресу пишем 14h и все - нага нет

Далее AC_Crackme_02_A.exe

Переход по адресу 401D75 осуществляется только если пароль равен "ExDec_Roxx". Следовательно пароль "ExDec_Roxx".

4 крякми AC_Crackme_02_B.exe:

401CEE: Dim var_CC As Integer = 982
401CF6: здесь будет что-то вроде:
If (var_A4 = var_CC) Then GoTo loc_401D25

var_A4 - то что мы ввели, var_CC = 982
Данный переход выполняется, если пароль неверный:
If (var_A4 = var_CC) Then GoTo loc_401D25
Следовательно пасс 982

Еще один крякми AC_ExDec_03_B.exe:

401FE0: push 0 'Byte
401FEC: Call 0(arg_3)

Думаю 401FEC - функция разблокировки кнопки. Предварительно видим:
401FE0: push 0 'Byte
0 - False, меняем на FF - True. Кнопка должна появиться.

Остальные крякми нормал - поисследую на предмет кейгенинга
