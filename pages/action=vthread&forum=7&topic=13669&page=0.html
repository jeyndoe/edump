Решил потестить RAMDisk "Enterprise" от QSoft.
Лень было искать тесты, написал батнег.

Code:
  2. @echo off
  3. if exist TEST.DAT goto :TEST
  4. echo Please wait. Creating 128Mb file for test.
  5. echo TheFuckingTest>TEST.DAT
  6. for /l %%i in (0,1,22) do type TEST.DAT>>TEST.DAT
  7. :TEST
  8. echo Copying 128Mb file.
  9. set time1=1%time:~-5%
  10. copy TEST.DAT TEST.TMP >nul
  11. set time2=1%time:~-5%
  12. if %time2:,=% lss %time1:,=% (set /a "time2=%time2:,=%+6000")
  13. set /a "timeZ=%time2:,=%-%time1:,=%"
  14. echo Time elapsed %timeZ% ms
  15. set /a "timeZ=1280000/timeZ"
  16. echo Speed %timeZ:~0,-2%.%timeZ:~-2% mb/sec
  17. del TEST.TMP
  18. pause >nul


Результат:

HDD:
Time elapsed 323 ms
Speed 39.62 mb/sec

RAM:
Time elapsed 44 ms
Speed 290.90 mb/sec

На рам диске от 200 до 300, возможно зависит он нагрузки проца или кеширование хз.
