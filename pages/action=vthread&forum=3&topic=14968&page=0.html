This is a tool for patching the protected or packed VB applications.

How it works?
VB applications use "MSVBVM60.DLL" to run.
Instead of unpacking protected or packed VB application we can inject our desire code into "MSVBVM60.DLL" to patch Main application.

So it produce "patched MSVBVM60.DLL" to be in the application directory.

It supports Armadillo,Themida,Execryptor,Private exe Protector,VMProtect,...
