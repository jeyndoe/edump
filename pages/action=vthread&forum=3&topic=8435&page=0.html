Имхо интересная идея
By AvAtAr from {RES}
Shows the 2 strings to be compared with the "__vbaStrCmp" function in Visual Basic applications, everytime that the application calls the function a window will show up with the strings that are gonna be compared

This project started because some friends were in the need of a proggie that was coded in VB, every version (lets say cd with a cost of $600) uses different serials and the serials change for each pc, besides the app uses about 7-8 modules (with different keys and costs $100 each module), since "vbaStrCmp" is used a lot to compare the serials i thought that would be nice to code something to make the serial fishing easier.
