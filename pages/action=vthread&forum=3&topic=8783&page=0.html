Punchit is a Freeware tool for bundling a music file with virtually any new or existing host executable (your application) to produce a NEW executable package that plays music. It does this by integrating the host executable and music file into a wrapper, thus freeing the developer from incorporating the music file into the application programmatically. Many different music types are supported. Punchit works with most packed / encrypted executable files. PunchIt relies upon such 3rd party tools as the BASS Audio Library for music playback and the PECompact2 Student Version for compressing your bundled application. In just a few easy clicks, you can enhance your application with the rich sound of music.
