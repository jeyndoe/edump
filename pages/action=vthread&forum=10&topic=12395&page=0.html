программка довольно простая... вот вчера заюзал для создания активатора...

d0wn, пользоваться её просто... смотри:
с оформлением я думаю всё понятно - тут ничего объяснять не буду...
итак секция Patch Memory - это для создания лоадера. при чём может использоваться напару со снифером. если файл нужный нам файл (выбирается нажатием кнопки Browse в секции Application) запакован, то для начала надо его анпакнуть и вписывать байты анпакнутого файла. вписываются только те байты, на которые нужно заменить.
теперь самое главное. вылавливание серийников. лично я тестил на Audio MP3 Maker v1.16 - там очень просто подсмотреть серийник. для начала кидаем программулину в отладчик и находим адрес, где сверяются введённый серийник и правильный. правильный серийник помещается в некую область - это мы всё знаем. смотрим адрес и вписываем его в serial sniffer в секцию Address [VA] и выбираем тип - eax, ecx и т.д.... собственно всё... заполняем все необходимые поля, жмём Create SS и получаем сниффер серийников. кидаем его в папку с прогой и тестим =)

з.ы. если всё же не совсем понятно, то могу сделать что-то типа тутора с картинками =)
