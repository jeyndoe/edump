Для расшифровки текста нужно с каждым байтом потока провести операцию:
Y:=X xor Key;
,где X - очередной байт потока
Y - результат(записать на место X байта)
Key - ключ
Этот самый Key представляет собой байт, который можно получить на основе поля ID, той таблицы откуда извлекаются данные.
Для этого содержимое ячейки поля ID нужно представить в HEX виде и взять младший байт - это и будет ключ. Теперь все
данные могут быть представленных в понятном виде.
