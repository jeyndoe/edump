YouTube Video Grabber is the software tool you can use to save online video from popular sites, such as YouTube, Google Video, MySpace, iFilm, Break, PutFile and others. The list of supported services is constantly growing, and since it is realized as the set of plug-in modules, you can easily download new plug-in from official site or either request for a plug-in supporting new internet service. This software offers online internet update service, so you will always have the last version of the program with the latest plug-ins. YouTube Video Grabber not only helps you to save online video , but you can also convert online video to a wide range of different formats, such as AVI, 3GP, 3GPP, WMV, DivX, Xvid, WMV, MPEG-1, MPEG-2, MPEG-3, MPEG-4. You can save just audio track of those video files to audio file (MP3, WMA, M4A, AAC, RA). Interface is straight and easy to use, so basically all you need to do is to enter URL of the video clip you want to save, set the folder for the output video file, choose the desirable output video/audio format and press Start button. For those people who have accounts on YouTube this software offers auto-login to save files which require to be logged in to access them. YouTube Video Grabber has another remarkable feature - the ability to search embedded video on any specified page. The program can find all the supported formats of embedded video and offer you to choose which clips to download or convert. You can also convert earlier downloaded files to any of supported format. Batch conversion is also supported, it makes downloading and conversion of a large amount of files much easier. You can find all necessary program settings on the main window. Thus, YouTube Video Grabber is the program for those people, who wants to save online video they liked and be able to watch it on computer, mobile phone, PocketPC, PSP, Zune etc.
