Из спецификации PDF:
Encryption-related information is stored in a document’s encryption dictionary,
which is the value of the Encrypt entry in the document’s trailer dictionary (see
Table 3.13 on page 73). The absence of this entry from the trailer dictionary
means that the document is not encrypted. The entries shown in Table 3.18 are
common to all encryption dictionaries.

Смотрим что там в этом "trailer"

<</Size 292/Root 257 0 R/Encrypt 289 0 R/Info 255 0 R/ID[<82E109B7E24B27EC00B67A565A14D8C4><3FCCDBBD8A04BB43AAB7D9609334C6 70>]>>

Объект 289 0:
289 0 obj<</Filter/Neovue:TrustViewSecHandler/V 1 ...

Требуется внешний плагин под названием "Neovue", поиск показывает что это e-Book content management platform, т.е. какая-то посторонняя хрень для защиты pdf и не только, поэтому стандартные брутфорсеры здесь не помогут.
