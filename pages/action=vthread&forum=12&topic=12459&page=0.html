v1.25
- fixed a bug where calc was creating an empty .ini file. The .ini is now
only read, never written. Thanks HandMill.

v1.24
- added sqrt() and constant "e"
- added various log functions: ln, loge, log for base e; lg, log10 for base 10;
lb, log2 for base 2; as well as log(base, x)

v1.23
- added user-defined output - add an "[output]" section to the .ini file and
specify output in a printf()-like fashion. Note, I only allow for one "%"
specifier in the format string. The expression can use any operators and
functions supported by the calc. There are three "vars" that are allowed
right now: $32s, $32u and $float.

[output]
# this gives "IPv4=192.168.1.2" when 0xc0a80102 is entered
IPv4=print("%u.", ($32s >> 24) & 0xFF); print("%u.", ($32u >> 16) & 0xFF); print("%u.", ($32u >> 8) & 0xFF); print("%u", $32u & 0xFF);

- added floor(), ceil() and fmod() functions that mimic their C counterparts:
the biggest int less than x, the smallest int that more than x and floating
point modulo.

[output]
# this one gives "weight=2 lb,10 oz" when 1.2 is entered
weight=print("%.f lb,", floor($float / 0.4536));print("%.f oz", floor(fmod($float, 0.4536)/0.4536*16));

- added a symbolic constant "pi". EG: sin(pi/2)

v1.22
- fixed hex stream input - "01 02 03" now generates a valid 32 bit int

v1.21
- fixed binary output - it now takes full 32 bits, unless big numbers
start
