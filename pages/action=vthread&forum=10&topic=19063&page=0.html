skypeaful writes:
@yagello: I can't use i12comp with multiarc (plugin for Far Manager). Can you help me how to setup and configure MultiArc + i12comp in Total Comander or Far Manager in English (I used Total Comander 7.03, Far Manager 2.0.1807)
for FAR - Edit custom.ini file (%FAR%\Plugins\MultiArc\Formats\Formats\Custom.ini), insert these lines
Code:
  1. [InstallShield12-14-15-16] 
  2. ID=49 53 63 28 B0 04 00 02, 49 53 63 28 78 05 00 02, 49 53 63 28 DC 05 00 02, 49 53 63 28 40 06 00 02 
  3. IDPos=0 
  4. Extension=hdr 
  5. Archiver=i12comp.exe 
  6. List=%P l -o -f %AQ 
  7. Format0="tt-dd-yyyy hh:mm zzzzzzzzzz aaaa pppppppppp nnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnnn" 
  8. Extract=%P e -o -d -f %AQ %FQ %FWQ 
  9. ExtractWithPath=%P x -o -d -f %AQ %FQ 
  10. Delete=%P d -o -d -f %AQ %FQ 
  11. Add=%P a -o -f %AQ %RQ %FQ 
  12. AddRecurse=%P a -o -r -f %AQ %RQ %FQ

as stated on ru-board forum, last page
