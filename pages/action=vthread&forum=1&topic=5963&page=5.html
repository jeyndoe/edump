1) Окно при входе в крякмис. Убирается так: по адресам заменяем данные значения на эти, 0x00004DC4--->00;0x00004DC5--->00;0x00004DC6--->A0;0x00004DC7--->E1.
Там у нас команда с функцией вызывающее окно о триале, мы эту команду данными изменениями делаем пустой.(проще говоря, будет выполняться функция с командами 'ни чего не делать')

Окно это можно убрать приведенным выше способом найдя эту команду с функцией, а можно испортить эту команду с функцией приведя ее к вылету из крякмиса, нам то все равно нужно выйти из него, какая разница как? По этому адресу крякмис смотрит текс из *.Rcs файла по иду, мы изменяем это значение, чтоб крякмис не нашел искомые значения или нашел ,но ошибочные, для последующего вылета из программы.
