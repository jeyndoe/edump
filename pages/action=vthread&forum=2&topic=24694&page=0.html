PUSH 1 - этот параметр для функции EnableWindow отвечает за 0 - выключение и 1 - за включение окна.
PUSH DWORD PTR DS:[402300] - выше по коду идет получение хэндла нужной нам кнопки:
Code:
  1. 00401083  |. 68 2C010000    PUSH 12C                    ; /ControlID = 12C (300.)
  2. 00401088  |. FF75 08        PUSH DWORD PTR SS:[EBP+8]  ; |hWnd = 00401000
  3. 0040108B  |. E8 02020000    CALL GetDlgItem     ; Получаем хэндл кнопки
  4. 00401090  |. A3 00234000    MOV DWORD PTR DS:[402300],EAX  ; хэндл который после вызова функции в EAX,
  5. ; заносится в DWORD PTR DS:[402300] его и будем подставлять как параметр для EnableWindow
