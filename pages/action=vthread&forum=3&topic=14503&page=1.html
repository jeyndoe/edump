Список изменений:
[31.05.2009] – 0.7
+ Добавлена возможность создания зашифрованных CD/DVD.
+ Добавлена опция 'load boot disk MBR' в список действий при неправильном загрузочном пароле.
+ Добавлена поддержка загрузки с зеркального диска программных RAID-1.
+ Добавлена поддержка файловой системы exFAT.
+ Достигнута полная совместимость с Windows 7 RC0 и RC1.
+ Улучшено определение имени SCSI дисков.
+ Добавлена реализация AES оптимизированная для процессоров с малым кэшем инструкций.
+ Добавлена поддержка аппаратного ускорения AES в процессорах VIA.
+ Добавлен инсталлятор.
+ Добавлены новые ключи командной строки.
+ Более стабильные результаты бенчмарка.
+ Добавлено автоматическое удаление точек монтирования при размонтировании, если они были добавлены при монтировании.
+ Добавлена поддержка дисков с большим размером сектора.
+ Добавлена корректная подпись всех исполнимых файлов доверенным сертификатом, за что спасибо ReactOS Foundation.
+ Мелкие багфиксы.
