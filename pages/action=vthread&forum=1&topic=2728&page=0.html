Для большей ясности (письмо от того самого чела, с кем и ведутся дебаты) :

********************************************************************** ********
Высылаю Вам программу установки "Компании". Если не получите
какие-нибудь части - сообщите, перешлю еще раз.

Процесс установки такой же, эта программа установится поверх демо-версии.
Единственная просьба - перед установкой программы желательно удалить
каталог с базой данных, который остался от демо-версии. Иначе не
гарантируется безопасность базы данных.

Чтобы узнать, где находится каталог с БД, запустите BDE Administrator
(по-умолчанию устанавливается в "C:\Program Files\Common Files\Borland
Shared\BDE\bdeadmin.exe"). На вкладке "Databases" выберите "ACompany" и
посмотрите куда указывает параметр "PATH". Например, если в "PATH" стоит
"C:\Program Files\ACompany\Database", то надо удалить папку "Database",
которая находится в "C:\Program Files\ACompany".

После этого установите программу с помощью "ACRemote4.0.exe". После
установки следует отправить нам для регистрации файл "acompany.exe" из
папки, куда установилась программа (по-умолчанию "C:\Program
Files\ACompany"). После регистрации этот файл будет отправлен Вам
обратно и его надо будет записать на место того, который Вы отправляли
нам (заменить).

********************************************************************** *******
