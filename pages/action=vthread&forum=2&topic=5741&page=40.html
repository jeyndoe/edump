Цитата одного из сайтов
Если ваш компьютер находится в локальной сети и получает доступ в Интернет через брандмауэр, вполне возможно, что часть сервисов глобальной паутины вам недоступна. HTTP-Tunnel позволяет разрешить эту ситуацию. Принцип его действия состоит в следующем: HTTP-Tunnel работает, как SOCKS-сервер. Программы на вашем компьютере передают данные HTTP-Tunnel, он перенаправляет их по стандартному порту HTTP (80) на один из серверов HTTP-Tunnel.com. Этот сервер отправляет данные реальному получателю. Аналогичным образом данные путешествуют и в обратном направлении. Таким образом, становится возможным использовать всевозможное программное обеспечение: Morpheus, ICQ, MSN, Kaazor, клиенты FTP, PC Anywhere — даже в том случае, когда в вашей локальной сети закрыты все порты, кроме HTTP.
Обрати внимание на ...Программы на вашем компьютере передают данные HTTP-Tunnel, он перенаправляет их по стандартному порту HTTP (80) на один из серверов HTTP-Tunnel.com...
Проверка валидности ключа осуществляется на сервере, только после этого пойдет перенаправление твоих пакетиков. Поэтому если твоего ключика нет у них в базе, то fullspeed ты не получишь.
