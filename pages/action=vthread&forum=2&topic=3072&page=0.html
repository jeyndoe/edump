Для того чтобы отключить ограничения ознакомительной версии программы, то есть сделать ее полнофункциональной, следует запустить Reg Organizer с ключем m (т.е выполнить команду organizer.exe m), затем в поле внизу окна ввести текст "exUSSR регистрация", после этого через пробел с большой буквы указать текущий месяц (по-русски) и нажать OK. Пример: "exUSSR регистрация Май".
