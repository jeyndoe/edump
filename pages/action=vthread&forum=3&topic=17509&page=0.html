[0utC4St], GUI для такой утилиты на фреймворке - зло. Можно ведь добавить оригинальную консольную версию в шелл:
Code:
  1. Windows Registry Editor Version 5.00
  2.  
  3. [HKEY_CLASSES_ROOT\*\shell\TrID]
  4. @="TrID Analyze!"
  5.  
  6. [HKEY_CLASSES_ROOT\*\shell\TrID\command]
  7. @="c:\windows\system32\trid.exe "%1" -w -r:10"

Некоторые работу с TRiD'ом автоматизируют, если загуглить, то можно найти нечто по теме ;)
