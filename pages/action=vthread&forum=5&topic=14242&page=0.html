Редмондский гигант всегда проявлял активный интерес проблеме лицензионности программного обеспечения. Корпорация намерена помочь защищать программное обеспечение сторонних производителей. Для этого была разработана технология BlockBox в настоящий момент не включенная в публичные бета релизы новой операционной системы Windows 7.

«При помощи специального дополнения к компиляторам VisualStudio разработчики смогут добавлять зашифрованные секции в свои Windows или .NET приложения. Дополнительные секции могут содержать любые данные и исполняемый код. Расшифровываться эти блоки будут при помощи персонального ключа составленного из лицензионного ключа Windows, данных полученных из Activation Server (система активации и проверки лицензионности Windows Vista. прим. ред.) и данных производителя ПО. Тем самым мы привяжем ПО к конкретному компьютеру и дадим огромную свободу возможностей нашим партнёрам. Для пользователей всё будет прозрачно: пока нет ключа не работает функциональность из зашифрованных секций.», рассказывает о принципах работы руководитель рабочей группы BlockBox (BlockBox Working Group) Айман Абадили.

Продажей программ защищённых BlockBox займутся авторизированные партнёры Windows Marketplace. Возможность использовать защиту Windows BlockBox для разработчиков будет предоставлена как опция некоторых MSDN подписок.
