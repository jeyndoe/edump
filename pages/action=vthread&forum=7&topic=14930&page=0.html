SLV пишет:
А как программа будет отправлять сообщения без пароля по протоколу все ищут меня?

Насколько мне известно, она предназначена только для удаленного управления с конкретного номера ICQ. На тот номер она и отправляет отчеты о выполненых командах. Я одно время её использовал, чтобы узнать динамический IP адрес компа, с помощью дополнительного плагина. Вот основной набор команд (без плагинов):

ICQ PC Control 2.2
==================
Доступные команды:
help - показать эту справку
time - время и дата на компе, аптаим
mes [сообщение] - показать сообщение
power [параметр] - управление питанием
компа
winamp [параметр]- управление плеером
winamp
tasklist - список запущенных процессов
taskkill [pid] - завершение процесса с
указанным PID
con - выполнение консольных команд
run [параметр] - быстрый запуск заданных
команд

Возможные параметры команды run:
list или без параметра - показать список
заданных команд
reload - перечитать файл runlist.txt
[1..9] или [имя] - запуск команды по номеру
или имени

Возможные параметры команды power:
monitor - выключить монитор
scrsav - запустить скринсеивер
reboot - перезагрузка
logoff - выход из системы
off - выключение

Возможные параметры команды winamp:
pause - пауза
play - старт
stop - стоп
next - следующий трек
prev - предыдущий трек
off - закрыть винамп
