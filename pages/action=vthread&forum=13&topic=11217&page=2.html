1. распакуй файл при помощи AspackDie
2. загрузи распакованный файл в Restorator и сохрани ресурс RCDATA -> TF
то же проделай при помощи PE Explorer
3. получишь в 1-м варианте сразу кое-что из файлов (например, звуки - Tf_FS2_StoredData.raw, убери в заголовке несколько байт при помощи WinHex, так чтобы файл начинался с RIFF, т.е. убери первые 93 байта и сохрани как водопад.wav)
а шо касаемо текста - во 2-м варианте получишь большой (10,8 МБ) файл TF.dfm, открой его как текст, копируй блоки цифр в Блокнот, убирай Enter'ы и пробелы, выделяй и Ctrl+C,
вставляй в WinHex так:

Edit -> Clipboard data -> Paste -> OK, выбери ASCII Hex и сохрани с расширением .htm
