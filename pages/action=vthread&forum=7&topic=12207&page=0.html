web-based torrent клиенты:
* torrentflux http://www.torrentflux.com/ (сам юзаю) [2.4 http://downloads.sourceforge.net/torrentflux/torrentflux_2.4.tar.gz?modtime=1213797423&big_mirror=0 ] - screenshots http://www.torrentflux.com/screenshots.php
* w3btorrent http://www.thewulong.com/w3btorrent/ (сегодня попробовал, работает) [0.1.4 beta2 http://downloads.sourceforge.net/w3btorrent/w3btorrent-0.1.4beta2.tar.gz?modtime=1210612414&big_mirror=0 ] - screenshots http://sourceforge.net/project/screenshots.php?group_id=154506 , demo http://www.thewulong.com/w3btorrent/demo/
* torrentvolve http://sourceforge.net/projects/torrentvolve (тоже пробовал юзать) [1.4 beta http://downloads.sourceforge.net/torrentvolve/torrentvolve-1.4-beta.zip?modtime=1167977557&big_mirror=0 ] - screenshots http://sourceforge.net/project/screenshots.php?group_id=179905

Torrentflux долгое время не обновлялся, последняя версия была выпущена в далеком 2004 году, хотя и работает стабильно и по сей день, но вот на днях (6/18/2008) вышла новая версия этого скрипта - 2.4:
* Powered by BitTornado 0.3.18 to support encryption
* Removed TorrentSpy search plugin
* Cryptography functionality & enabled by default (by imrook)
* Added checks to prevent normal users from seeing sensitive information (by deadeyes)
* Updated MiniNova search plugin (by alx)
* Removed CVS files for BitTornado
* Converted \r\n to \n in all source files
* Bug fix for torrents dying after they're started (tornado shell args not escaped)
* Postgres 8 sql install scripts
* Updated MySQL install scripts (TYPE clause depreciated for storage engine declarations)
* Fixed bug preventing torrents from starting in simple mode

Bug fixes from Debian team
* Recursive call error in SearchEngineBase
* Python filename dependency
* Parameter check in directory browser to prevent traversal above download directory
* Update settings fails with blank search filter

Плюс ко всему в интернете есть модификаия старого TorrentFlux'а, называется - Torrentflux-b4rt http://tf-b4rt.berlios.de/ , руки никак не доходят поставить и поюзать.

Требования:
- TorrentFlux
* A Unix like OS (Linux, BSD, Solaris, etc.)
* A Web Server that supports PHP (e.g. Apache)
* SQL Database (MySQL, Postgres)
* PHP 4.3.x or higher
* Python 2.2 or higher
Note: Some add-ons may have additional requirements

- w3btorrent
* Unix-like server
* Apache, lighttpd or AppWeb (AppWeb has some issues)
* PHP5+
из модулей:
* Ctorrent
* Nohup
* Crontab
* Wget
* Tar
* Gzip
* Bzip2
* Rar
* Zip
* Unzip
