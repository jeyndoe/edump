DUSuperControler v1.9

Утилита, которая поможет эффективно распределить трафик между пользователями и приложениями. Она выводит список всех приложений, запущенных на компьютере, и дает возможность для каждого из них определить максимальную скорость загрузки и передачи файлов, а также общий объем переданных и принятых данных. Также можно задать все эти параметры не отдельно для приложений, а для каждого из компьютеров в сети.

Самый простой и удобный из опробованных шэйперов.
