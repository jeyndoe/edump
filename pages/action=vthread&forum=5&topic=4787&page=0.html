maikkri5ki пишет:
это файл еще упакованный(AsProtectom)! Тебе дамп прислать, что ли?
Нет уже не надо, дамп сделал. Только он тоже не запускается. И импортов мало. Скорее тут вот что: из пакера невозможно обнаружить функции таким образом правильно. Надо смотреть в любом HEX редакторе адреса и искать там таблицу импорта. например по функции GetModuleHandleA или ещё по какой нить. Потом когда найдёшь надо с помощью PE Editora выбрать дамп и нажать FLC. Потом вычислить RVA (с помощью offset). Потом удалить все функции с надписями NO и сделать Fix Dump.
