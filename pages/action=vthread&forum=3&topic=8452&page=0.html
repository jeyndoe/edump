Hpmbcalc is a programmable multiple-precision hex calculator with many utilities. You can use Hpmbcalc to perform a lot of big number math operations such as: square root, modular inverse; hash and CRC calculation; pseudo-random number generation; primality test; hex, dec, oct, bin conversion; etc.
***** What's new? *****

1. Added: The ability to calculate 64-bit checksum.
2. Added: The ability to exclude certain byte-ranges from a file
for checksum/CRC/hash calculation.
3. Added: Copy operand and trim spaces commands.
4. Added: Checksum calculator button on the toolbar.
5. Improved: Some tools and online help documents.
