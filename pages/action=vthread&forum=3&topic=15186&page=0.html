После того, как появилась проблема с ТруКриптом (частые бсоды, по совершенно непонятной причине), решил начать поиск альтернативы ТруКриптом. основными критериями поиска были портабельность (работа без установки), возможность создавать зашифрованные файлы-контейнеры, бесплатность, открытость исходного кода. Поиск закончен, результат успешен и достаточно интересен, итак.

FreeOTFE

Поддерживаемые O/S: Windows 2000/XP/Vista, Windows Mobile 2003/2005, Windows Mobile 6

FreeOTFE это криптографическое приложение (OTFE – On The Fly Encryption) для создания зашифрованных файл-контейнеров и шифрования разделов, сравнимое по возможностям с TrueCrypt, включая поддержку "portable mode", "plausible deniability" и отсутствие "volume signature".

+ Бесплатное (Free) ПО с открытым исходным кодом.
+ Поддерживает форматы некоторых Linux-криптоконтейнеров – Сryptoloop "losetup", dm-crypt, LUKS.
+ Есть возможность добавления новых алгоритмов шифрования в виде отдельных модулей-драйверов.
+ Версия программы для PDA также может работать с криптоконтейнерами, созданными на PC.

Официальный сайт

распараллеливание в TC:

When your computer has a multi-core processor/CPU (or multiple processors/CPUs), TrueCrypt
uses all of the cores (or processors) in parallel for encryption and decryption. For example, when
TrueCrypt is to decrypt a chunk of data, it first splits the chunk into several smaller pieces. The
number of the pieces is equal to the number of the cores (or processors). Then, all of the pieces
are decrypted in parallel (piece 1 is decrypted by thread 1, piece 2 is decrypted by thread 2, etc).
The same method is used for encryption.

So if your computer has, for example, a quad-core processor, then encryption and decryption are
four times faster than on a single-core processor with equivalent specifications (likewise, they are
twice faster on dual-core processors, etc).
