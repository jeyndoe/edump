Почти забытый сегодня Netscape Navigator, который в начале интернетовской эпохи был абсолютным лидером среди браузеров, не только не исчез, возродившись не так давно в восьмой версии, но и продолжает активно развиваться, обретя и множество новых возможностей. Одна из самых заметных среди них — многоплатформность. К новшествам можно отнести и появление средств социальной интеграции с другими пользователями, инструментарий для работы с лентами новостей RSS, систему автоматической коррекции вводимых адресов и средство временного хранения ссылок для дальнейшего просмотра, не захламляющее список постоянных закладок.

AOL has released a new of Netscape Navigator - Speed, Flexibility and More Security Choices Than Any Other Browser. Based on Firefox, the Netscape Browser options and interface are similar, but brought to a superior level of accessibilty by filling some of the Firefox gaps, such as tabbed browsing, context menus and menu bars. The new Netscape Browser test build runs on two different browsing engines: the Mozilla Foundation's Gecko engine, which powers up the Mozilla, Firefox and older Netscape browsers, and Microsoft's Internet Explorer engine.
