Гм, а что в версии 2.0 code replace`ом стали называть то что раньше было 'Embedded protector' ? Code replace эте пятибайтовые куски которые тырятся по всему коду и которые можно восстановить в оригинале 1 в 1, либо просто переделать процедуру их обработки.
