Для того, чтобы сделать невозможной отладку StarForce использует те же ресурсы, которые нужны SoftICE - он не только затирает DRx регистры, но и использует их для работы, он перенаправляет обработчики прерываний, которые юзает SoftICE, и тоже их использует для своей работы, участки кода выполняются в pcode. Это писалось на форуме reng.ru. Поэтому использовать SoftICE весьма затруднительно. Для взлома StarForce необходимо написание вспомогательных прог.
