Надо заменить обои и значки в прошивке, а также есть предположения что прошивке скрыты некоторые функции, есть одна проблема все зашифровано, я только смог распаковать прошивку.
1. Распаковал *.ipsw с помощью WinRAR
2. Там оказалось 2 файла (Firmware.mse и N20.bootloader.release.rb3)
3. Firmware.mse распаковал с помощью extract2g (http://rghost.ru/40553335)
4. Появилось 9 файлов формата *.fw (их можно залить на устройство по отдельности)
appl.fw "предположительно лого"
bdhw.fw
bdsw.fw
chrg.fw "предположительно charge"
diag.fw "предположительно диагностика"
disk.fw "предположительно связанное с памятью"
lbat.fw "предположительно Low battery"
osos.fw "предположительно программный код"
rsrc.fw "предположительно ресурсы"
Возможно превратить в образ диска (*.bin) с помощью DD (http://rghost.ru/40553559)
Может кому нибудь поможет nanotomy (http://rghost.ru/40553604)
Только я незнаю как, там все зашифрованно, 3 день бьюсь.
Прошивку можно скачать отсюда (http://appstudio.org/ipod/) выбираете версию 1.2 ipod nano 6g картинка в аттаче.
