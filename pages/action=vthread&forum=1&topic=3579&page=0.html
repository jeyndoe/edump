Hi, The Old Pirate
I found a problem:
TC Power Pack you installed contains extra sound theme, you can unselect it while installation (last option - 'Звуковая тема')
if you want to remove these sounds:
- goto to Control Panel -> Sounds and Multimedia -> in sounds tab find sound defined for TotalCommander, set all TC related sounds to 'none', or switch to 'No sounds' sound theme
- goto %WinDir%\Media and delete .wav files that begins with 'TC' like 'TC Delete Complete.wav'

batch command for this:
del %WinDir%\Media\TC*.wav
