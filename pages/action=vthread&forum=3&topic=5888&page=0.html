Описание от разработчика
VB RezQ provides an accurate framework on which you can rebuild your VB application. It identifies all the source files, recovers the project file, the graphical design of each form, references to custom controls and declarations for API calls. It also identifies all events and subroutines and embedded resources. Note: VB RezQ does not recover the Basic code within events, functions and subroutines.
Find out more in our overview http://www.vbrezq.com/oview.php .
