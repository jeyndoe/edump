А если есть?
Пароль в БИОС держит батарейка – вытащили ее сняли пароль, но на диск пароль остался – значит он записан на самом жестком (где то в системной области). Вот надо его как то вытереть.

Тут www.datarecovery.com.sg/data_recovery/hard_disk_lock.htm последний абзац.

«… In Maximum security mode, one can not unlock the disk without knowing the passwords. One way to reuse the disk is to issue the SECURITY ERASE PREPARE command followed by SECURITY ERASE UNIT. However, The SECURITY ERASE UNIT command will require the Master password and all data will be erased as a result.»

Т.е. есть команды которые сбрасывают пароль с потерей данных.
