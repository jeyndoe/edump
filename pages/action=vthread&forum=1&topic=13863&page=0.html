licsrv - известный демон от одной известной фирмы L.....K или еще H..N. SID-ы для него не подобрал только супер ленивый без интернета. Для Flex v7 без CRO это делается по Woodman-у 1в1 без всяких заморочек. Почитай и сделай нормальный генератор. Фичи возьми с Сановской машины. Если хочешь поразбираться ессно.
