Когда Вы меняете переход по указанному адресу, в папке с программой создаётся файл code.txt с правильным кодом для Вашего ID. Если у Вас сомнения в корректности взлома, удалите файл code.txt, предварительно скопипастив код, перезапустите программу, на этот раз ничего не трогайте в памяти, а просто вставьте код. Будет поздравительное сообщение, а раз ничего не изменялось в памяти, код для текущего ID'шника верен, значит вопрос насчёт test-version имеет смысл задать автору программы.
