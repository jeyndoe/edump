Зачем такие трудности?! Скачиваете последний HlpViewer 2.3 который вырезал из студии 2017, архив 29 Mb --> Link <--

Устанавливаете в порядке -
1.Microsoft.HelpViewer,version=2.3.0.28
2.Microsoft.VisualStudio.Help.Configuration.Msi,version=15.0.26621.2,language=en-US
3.Microsoft.VisualStudio.MinShell.Msi,version=15.0.26711.1
4.Microsoft.VisualStudio.MinShell.Interop.Msi,version=15.0.26621.2
5.Microsoft.VisualStudio.MinShell.Msi.Resources,version=15.0.26621.2,language=en-US

Далее копируете ярлык HlpViewer - Shortcut.lnk на рабочий стол, запускаете, должно получиться так



Скачиваете всё самое нужное, меняете путь к своему архиву, все файлы автоматом скопируются в новое место, пользуетесь оффлайн Архив можно таскать где угодно, главное повторить новую установку хелпера на новой системе и указать ему путь на папку с книгами. Эта версия не требует установки Visual Studio 2017, хелпер работает автономно, модифицирован только ярлык запуска.

Для тех кто уже установил Visual Studio 2017 или хочет вызывать из студии через меню хелп вызывают HlpViewer 2.3, скачивают контент один раз в любимую папку и используют дальше оффлайн. Оба варианта как со студией так и без отлично работают.
