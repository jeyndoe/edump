Качаем скрипт на сервер, распаковываем к какой-нибудь каталог, смотрим конфиг, правим (при необходимости) сейчас скрипт поддерживаеи даже многопользовательскую работу, квоты, авторизацию, список прокси. Запускаем index.php заходим в закладку option, ставим галочки там (при необходимости), если есть Премимуму тоже можно их вбить туда, настройки сохраняться в кэше и все!

вбиваем линк на файл в первой закладке Main, при необходимости можно менять Refer, пароли, описание файла, жмем загрузить, вводим код на картинке, и загружаем файл.

Последняя закладка - закачка списка файлов, каждый файл - отдельная строка.

С закаченными файлами работаем в разделе Files, возможностей - туча!

Да, не все бесплатные хостинги терпят этот скрипт, многие позакрывали соксы, органичили размеры закачаваемых файлов
