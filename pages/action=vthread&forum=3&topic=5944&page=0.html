The Silicon Realms Toolworks is happy to announce that Armadillo 4.48 "Beta 1" test version is now available! The direct download link can be found here:

www.siliconrealms.com/armadillo_engine.shtml

The changes between version 4.44a and the beta consist of the following new features:
Added support for screen saver protocol "/p" to be used and suppress nag screens.
Made a change to the client/server network handling where if the 'locateserver' port gets closed due to an error, we will report the error and attempt to re-open it.
Other minor bug-fixes and other changes. Refer to Revision History for the list.
As always, we don't recommend releasing software with the beta version, but if you must do so, please test it thoroughly first to ensure that we haven't missed a bug.

If there are no major problems, the final release will be on August 14th 2006.

As always, please let us know if you have any questions, or run into any problems with it.

D080A010 Version 4.48 Beta 1 (Public Build)

www.siliconrealms.com/download/betas/armadillo.exe

New features:
· Added support for screen saver protocol "/p" to be used and suppress nag screens. (Case 14177)
· Made a change to the client/server network handling where if the 'locateserver' port gets closed due to an error, we will report the error and attempt to re-open it. (Case 14542)

Bugs fixed:
· Corrected the case where on reverting to Default on an invalid key would overwrite the "TRUE" in the InvalidKey environment variable with a NULL before it could be seen and acted upon by the wrapped program. (Case 14240)
· Added code to prevent function names longer than 512 bytes from overflowing the buffer and causing an exception. (Case 13127)

Other:
· Some random security changes/enhancements.
· Added a note in the help file to "General DLL and OCX files" about trial versions. (Case 13044)
