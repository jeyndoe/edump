Кто хотел научиться работать с USB в Delphi? Этот компонент позволяет отправлять и принимать данные с USB порта. Работа схожа с COM портом, поэтому данный компонент понадобиться для промышленников (тех, кто кодит проги для промышленных устройств, подключаемых по USB).

Компонент включает в себя DLL библиотеку, через которую и происходит обмен данными. Если найти доки, то можно попытаться написать прогу обмена данными с сотовым телефоном по USB.

Компонент включает полный исходник с запускным файлом и DLL библиотеку. Размер: 518к.
