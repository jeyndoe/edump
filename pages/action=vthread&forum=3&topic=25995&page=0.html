VirtualBox Hardened Loader - лоадер для предотвращения детекта виртуализации.
Любопытный проект, ценен как годный опенсурс с подробными манами по детекту для нуждающихся, как впрочем и виртуальная машина для которой сделан, там сырки вообще кладезь.

Системные требования x64 Windows 7/8/8.1/10
Версии VirtualBox от 5.0 до самой свежей (хотя по мне 6-ка еще не торт, остался на 5.2).

Для сборки нужны
Microsoft Visual Studio 2013 U4 или Visual Studio 2015/2017 + Windows Driver Kit 8.1 U1 или выше.

Частично может юзаться под линуксом, также на странице проекта есть ссылки по рекомпиляции VirtualBox.

Обратите внимание на особенности при компиляции подписанного драйвера.

Пару дней назад обновился под 6-ку, вот вспомнил.
