Thrasher
Попробуй bp GetLocalTime

А вообще для лучшего понимания, поставь пару раз бряк следующим образом:
Грузишь прогу в оллю->View->Executable modules(откроется окно, в столбце Name ищешь kernel32, потому что функция GetLocalTime вызывается из этой dll)->правой клавишей по kernel32 ->Viwe names (откроется ещё одно окно, щелкаешь по столбцу Name и набиваешь на клаве первые буквы GetLocalTime). Встаешь на искомую функцию и ставишь бряк F2(это соответствует bp GetLocalTime).
