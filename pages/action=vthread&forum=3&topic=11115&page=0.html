Если у кого-нибудь есть .NET Framework 3.5, скиньте, пожалуйста, в любой файлообменник такие сборки: System.Core, System.Xml.Linq, System.Data.DataSetExtensions, System.Data.Linq. Эти сборки нужны мне, в основном, для исследования.

Прошу потому, что весь пакет .NET Framework 3.5 скачать нет возможности (большой объём). Вышеперечисленные же сборки должны весить около одного мегабайта.

В Google искал - не помоголо.

Если кто не знает, после установки .NET Framework 3.5, сборки можно найти в папке: <корневая_папка_windows>\Microsof.NET\Framework\v3.5.<номер_билда>
