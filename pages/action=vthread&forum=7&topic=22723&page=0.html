Microsoft .Net Native
Анонс на Build 2014.

.NET Native compiles C# to native machine code that performs like C++. You will continue to benefit from the productivity and familiarity of the .NET Framework with the great performance of native code. Popular Windows Store apps start up to 60% faster and use 15-20% less memory when compiled with .NET Native.

Многие мечтали что бы .Net приложения работали быстрее и их нельзя было прочитать как открытую книгу на родном языке. Уже сейчас можно переводить Windows 8.1 Store приложения в Native code (Developer Preview), в планах компании так же значится расширить применения этого чуда и до Desktop приложений в будущем.

For developers, this means:
Your apps will provide the superior performance of native code.
You can continue to program in C#.
You can continue to take advantage of the resources provided by the .NET Framework, including its class library, automatic memory management and garbage collection, and exception handling.

For users of your apps, .NET Native offers these advantages:
Fast execution times
Consistently speedy startup times
Low deployment and update costs
Optimized app memory usage

И т.д., и т.п.

link: http://msdn.microsoft.com/en-US/vstudio/dotnetnative
