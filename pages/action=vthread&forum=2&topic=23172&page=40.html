Перед запуском программы.
1. Идешь в User AppData:

(путь в Vista и выше)
c:\Users\ИМЯ ЮЗЕРА\AppData\Roaming\Analysis Lotto\
(в XP путь)
C:\Documents and Settings\ИМЯ ЮЗЕРА\Application Data\Analysis Lotto

находишь файл ADLOTTO.INI, и во второй уже существующей строке пишешь:
9999(три пробела)100101
где последние 101 это количество запусков до истечения full версии, значение будет убывать при каждом закрытии программы.

2. если есть файл RN LOTTO.GEN по этому же пути - удаляешь, это первый маркер триала.
3. если есть файл DBCCY.EPF в папке WINDOWS - удаляешь, это второй маркер триала.

Когда лимит запусков исчерпается, маркеры запишутся.
Проделать вышеуказанные операции заново и триал сбросится, а программа превратиться снова в Full.
Поиграешся с последними значениями 101 - можешь выставить больше, но не больше тысячи.

http://rgho.st/7XpWcTFCP.view
