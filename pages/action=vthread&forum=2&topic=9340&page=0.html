Сайт hxxt://zinio.com/

Компания занимается платным распространением журналов и других периодических изданий, около 300 издательств распространяют свою продукцию через zinio.com более чем 2 млн. подписчикам. Представительство компании по всему миру.

Для публикации журналов компания разработала свой вариант PDF защищенного DRM, формат имеет расширение .ZNO, а также свой клиент Zinio Reader (скачать можно с их сайта после регистрации). Очень удобный, заточен именно для чтения журналов и книг.
Встроенный менеджер загрузки автоматически предлагает доступные журналы для загрузки.

Некоторый издания можно выписать бесплатно, после заполнения несложной анкеты, часть журналов можно скачать бесплатно (для затравки ). Существенные скидки при оформлении подписки. Существует web клиент Zinio Express, написан на flashе, часть журналов доступно через браузер тоже бесплатно.

Сохранить можно только по страничке через печать сраниц на виртуальный принтер.
Т.е. теоретически можно перегнать журналы в PDF только картинками.

В сети встречаются упоминания (204-2005 г.г) что чужие журналы можно смотреть при наличии оригинальной XML, которая содержит цифровую подпись и создается при легальном скачивании журнала, но у меня не заработало.

Технически можно скачать любой журнал .zno но вот открыть его не удастся, т.к. нужна цифровая подпись (файл формата XML), которая скорее свего формируется сервером на основе уникального IDшника клиента.
