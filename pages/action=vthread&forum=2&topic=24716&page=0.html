Для п.5 (см. ниже) необходимо указать компилятор(протектор) Вашей программы, который можно определить с помощью анализатора Detect It Easy (DIE). БЕЗ ЭТОГО ПУНКТА ЗАПРОС БУДЕТ УДАЛЕН.
Причем для DIE надо указывать не файл установщика (setup.exe, и т.д.), а сам исполняемый файл, который вы запускаете для работы с программой! Исключение - программа не может быть установлена без валидного серийника, т.е. взламывать необходимо сам инсталлятор.
