Установка полной версии программы, записанной на компакт диске.

1. Запустить файл "Ccalc6.80.exe" (двойное нажатие мышкой по нему) и на открывшемся окне
справа внизу нажать кнопку "извлечь". Программа автоматически установится и
на рабочем столе появится ярлык для запуска программы "Расчет Конструкций".
2. Запускаете программу ярлыком с рабочего стола. При первом запуске она у Вас
запросит регистрацию. На окне регистрации нажимаете кнопку "серийный номер сохранить
в файл", затем выбираете папку, куда Вы будете сохранять этот файл, набираете
произвольное имя файла (например "123") и сохраняете его.
3а. Если у Вас есть доступ к электронной почте, то этот файл нужно выслать
разработчику по электронной почте.
3б. Если у Вас нет доступа к электронной почте, то этот файл нужно распечатать
крупным шрифтом (поставить размер шрифта 24) и отправить разработчику по
факсу.
4. Обязательно при отправке файла с серийным номером сообщите: название
организации, город, номер рекламного телефона.
5. В ответ Вам прийдет файл с регистрационным номером. Нужно на окне регистрации
нажать кнопку "регистрационный номер открыть из файла" и выбрать этот файл
с регистрационным номером. Программа автоматически закроется.
6. Повторно запускаете программу, если регистрация прошла успешно, то программа
будет спрашивать только имя пользователя и пароль. Изначально заложено имя
пользователя "admin" без пароля.
