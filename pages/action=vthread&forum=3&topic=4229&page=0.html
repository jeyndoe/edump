Заходишь на виндовс абдейт с установленным SP2, проверяешь систему, записываешь номера обновлений (KBxxxxxx). Затем подставляешь в ссылку шесть цифр, без KB и качаешь обновление для своей оси. Сейчас их уже больше тридцати, так что для установки можно сделать батник:
Echo Off
KBxxxxxx.exe /quiet /norestart
exit

support.microsoft.com/?kbid=xxxxxx
...
Вместо xxxxxx номер SP без kb.
