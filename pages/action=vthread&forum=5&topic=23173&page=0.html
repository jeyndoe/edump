geograph значит так.

Назовем "HID.exe, HID-hdd.exe" - сборщиком.

Суть этих утилит - запуститься от имени пользователя с правами достаточными для получения заранее заданной информации о установленном на данном компьютере оборудовании. Обычно этой информацией является ID CPU, HDD, MotherBoard и другое редко заменяемое оборудование компьютера.

После получения этой информации сборщик "хитрым образом" использует полученную инфу, изменяет ее, приводит к другому виду. Вы высылаете ее разработчику, он знает этот "хитрый алгоритм" и может восстановить настоящие ID CPU, HDD, MotherBoard..., если конечно это не хэш функция (не обратимая).

В lic.license содержится инфа которая при запуске приобретенного ПО проходит обратный путь "хитрого алгоритма", и смотрит - "а на том ли компьютере запущено ПО которому выдана лицензия? Сравню ID CPU, HDD, MotherBoard...".

Вам в раздел "Все для .NET".
