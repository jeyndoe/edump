кинул, чтобы каждому не отписывать вот что я делаю, прошу указать ошибки:

1)в debugging options убираю все галки
2)жму f9 (попадаю в эксепшен)
3)22 раза жму shift+f9 (там будет большая пауза),а за ней эксепшн (последний ексепшн перед стартом проги)
4)смотрю куда хендлер сеха указывает и ставлю туда бряк, жму shift+f9 (адрес на 3ххххх)
5)оказываюсь на бряке
6)дальше по F7 трассирую пока не стану на RETN (выход в кернел)
/чтобы найти stolen code в предвдущих версиях прота:
|7)открыть закладку memory и бряк на 3ххххх(тот диапазон что seh) память break on access
|8)обратно в код и shift+f9 , убираю break on access с этой памяти, после пункта 9 stolen code будет в trace
\ log) в куче мусора.
9)break on access на 401000
10)ctrl+f11, брякаемся в коде проги (там будет переход в область памяти протектора)
11)поднимаемся чуть вверх там oep дельфийской проги
