Inoger "Путь" - в данном случае это это место на диске где располагается Файл или Папка. В данном случае: 
Run trace file=rtrace.txt
API help file=с:\OllyDbg\OLLYDBG.HLP
Text save file=
Symbolic data path=
UDD path=с:\OllyDbg
Plugin path=с:\OllyDbg
с:\OllyDbg\ ты должен изменить, если у тебя Олька стоит на D в папке CRACK, например: d:\CRACK\OllyDbg\
