v3.7.0 New :

What's new in this version:
Search engine:
added: ActiveMark 5 (used by www.trygames.com) keys search (avaiable for RTKF donators only);
added: ExeCryptor keys search routine;
added: ZipWorx SecureEXE keys search routine;
added: Trial Creator keys search routine;
added: SDProtector keys search routine;
added: search for invalid TypeLib keys;
added ability to delete "hidden" keys which may contain any number of nested subkeys (they can be "hidden" too);
retrieving content information from "hidden" keys.

User interface:
added German language UI (by Dirky);
added "classic" menu with more items;
added program documentation as HTML help file;
now Tooltip for keys found is more factual;
now 'Jump To Regedit' works simply super fast.

Command line:
added /j=any_reg_path switch to rapidly open specified Registry key in Regedit.

morenews.ru/cgi-bin/dbloadfile.cgi?post=8576&id=1
Registry Trash Keys Finder 3.7.1 (149Kb) удаляет из реестра Windows вредные ("мусорные") ключи, которые оставляют за собой некоторые программы. Также программа может помочь при продлении триальности некоторых программ защищенными ASProtect, Armadillo, ACProtect, SVK, Obsidium, ExeCryptor, ZipWorx, SDProtector, Trial Creator, TypeLib.
Что нового в этой версии:
- добавлен поиск ключей новых версии ASProtect 2.2 SKE / 1.35,
- добавлено: ключ командной строки /j="reg_файл" для открытия в редакторе реестра первого найденного в .reg/.key файле валидного пути к ключу реестра,
- в контекстное меню файлов .reg и .key добавлена команда "Открыть в Regedit", открывающая в Regedit найденный в файле путь к ключу реестра,
- теперь также допустимо использовать .key-файлы в папке для автовыполнения ("AutoREGs"),
- багфикс: "Run-time error '5'..." при поиске в ключе, содержащем некоторое значение с нулевым размером (это возможно только на системах на базе NT),
- багфикс: некорректная кодировка символов в окне просмотра содержимого ключей на некоторых Win9x системах,
- багфикс: ключи SDProtector могли быть продублированы в списке.
