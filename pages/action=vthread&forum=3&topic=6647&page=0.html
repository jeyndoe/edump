Если запустить без параметров. то увидим следующее:

Usage: CMP file1 file2 [file3 .. fileN] [-switch1 .. -switchK]

Switches:
-a Show differences if bytes in files have ascending order
-d Show differences if bytes in files have descending order
-f:filename Output file name, console by default
-o:######## File offset [PASCAL style]
-s[:###] Show if bytes in files change on a step [PASCAL style]

Notes:
If -a, -d and -s switches aren't set, all differences are showed
