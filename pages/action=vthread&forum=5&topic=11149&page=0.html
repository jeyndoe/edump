Читай Help.chm для начала.
Выбор игроков - прежде чем начать игру, выберите соперников. Если список игроков пуст, то создайте, как минимум, парочку игроков. При создании игрока требуется ввести его имя и изображение. .....skip......Изображение следует взять из какого-нибудь графического файла. После этого наметьте фрагмент для вырезания. Он будет автоматически пересчитан к пропорции 3х4.
