Смотри в сторону Ange Albertini (aka Corkami). У него масса таблиц по форматам PE, PE64, Mach-O, PDF, ZIP, ...
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/PE102posterV1.pdf
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/PE101-v20L.pdf
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/pe101-64b.zip
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/pe-20110117.pdf

Там же множество представленных семплов с минимальной функциональностью "Hello World", но содержащие одну из особенностей формата (delay imports, bound imports, recursive resource directory, calling its own exports, DLL with no exports, Imports forwarding, PE with 99 sections (ХР), PE with 199 sections (win7) и т.д.).
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/BinaryCorpus_v2.zip

Таблицы опкодов Java (JVM), .Net (CIL), Dalvik (DVM)
https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/corkami/opcodes_tables_complete-120221.pdf

Т.е. дядя поработал и сделал всю методработу за вашу кафедру.
