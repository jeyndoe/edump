Версия 0.93. Дата релиза: Январь 20, 2008 unrealcommander.org/download/uncomsetup.exe
Добавлено:
- Возврат после отключения FTP-клиента в последний посещенный каталог локального диска
- Возможность отключить перечитывание сетевого каталога при становлении программы активной
- Использование команды "cd" для FTP-соединений
- Имя текущего файла/каталога подставляется как вариант для создания нового каталога
- Опциональный вход в создаваемый каталог
- Округление размеров файлов/дисков до гигабайтов (целое и десятичное)
- Возможность указать буквенный диапазон при групповом переименовании
- Возможность одновременного переименования файла/файлов по маске при копировании
- Изменение размеров многих функциональных окон
- Удаление файлов при переносе в архив
- Поиск по выделенным файлам/каталогам
- Копирование даты/времени каталога (опционально)
- Смещение курсора на строку ниже при выделении пробелом (опционально)
- Возможность изменить тип отображения пути в результатах поиска (полный путь, путь от текущего каталога, без пути)
- При функции обновления ключа из удаленного каталога возможно обновить и exe-файл программы
- Функция быстрого соединения с FTP-севером
- Отраслевой вид (с подразделами)
- Повторение попыток соединения с FTP-сервером через заданный интервал времени
- Отправка заданий копирования/перемещения/удаления сразу в фоновое выполнение
- "Стандартный" формат отображения имен файлов/каталогов (первая буква большая, остальные маленькие)
- Отображение знака ">" для каталогов, расчет размера которых был прерван
- Отображение даты истечения срока действия ключа в окне "О программе"
- Возможность использования программы полностью без лицензионного ключа (или с просроченным ключом) в режиме ограниченной функциональности
- Функция автоматического обновления ключа за несколько дней до момента истечения срока действия имеющегося
- Режимы отображения списка файлов со стандартными значками и совсем без значков
- Функции копирования/переноса/правки комментариев файлов (descript.ion и files.bbs)
- Функция перехода в каталог/запуска файлов по одинарному клику
- Поиск файлов/каталогов с учетом атрибутов
- Возможность поиска по нескольким дискам
- Возможность использования копирования/переноса средствами ОС для конкретных дисков (например флэш-драйв)

Исправлено:
- Некорректная работа сочетания Ctrl+влево/вправо при нахождении в архиве
- Некорректная работа контекстного меню каталогов для архивных файлов
- Постоянное обновление информации о свободном месте на диске для FDD и сетевых каталогов
- Отсутствие запроса имени пользователя FTP-соединения если оно не было указано в настройках
- Возможность перехода к окну, перехода к которому быть не должно
- Удерживание Insert на последнем файле каталога при кратком режиме просмотра выдавало ошибку
